# Valeton GP-180 — русская документация

Практическая база по процессору **Valeton GP-180** (firmware V1.0.0): перевод инструкции, модули эффектов, SnapTone/NAM, пресеты для мандолин и акустики, узкий набор электрогитары под ансамбль, EQ в миксе бэнда.

**Лицензия и благодарность Valeton:** [LICENSE.md](LICENSE.md) (**CC0 1.0** — свободное использование в любых целях, в т.ч. компанией-производителем). Юридический текст: [LICENSE](LICENSE).

> **Электрогитара:** большинство заводских патчей GP-180 уже сделаны под электрогитару — полный разбор этого каталога в проекте **не ведём**. Пользователь может изучить factory-звуки самостоятельно. Мы готовим только **чистые** и **слегка перегруженные** электро-пресеты под ансамбль (бас · аку-ритм · электро · мандолина · ударные) в стилях **фолк / регги / dark americana** — см. [docs/presets/elektro-ansambl/](docs/presets/elektro-ansambl/).

Исходный PDF: [`source/GP-180_Online Manual_EN_Firmware V1.0.0.pdf`](source/).

## Цепь по умолчанию

```
NR → PRE → WAH → DST → N→S → AMP → CAB → EQ → MOD → DLY → RVB → VOL
```

До 12 модулей одновременно, порядок свободный. Сэмплирование 48 кГц, >200 эффектов, до 100 SnapTone (50 заводских), до 20 пользовательских IR, 200 патчей (100 factory).

## Быстрый старт

1. Инструмент → **IN**, выход **OUT L** → монитор / FX Return комбика (громкость комбика приглушить).
2. **MASTER** вниз → включить питание.
3. Тюнер: удержание **A+B**.
4. Выбор патча: футсвитчи **A/B** или крутилка **PARA**.
5. Поднять MASTER и громкость системы.

Подробнее: [Начало работы](docs/manual-ru/04-nachalo-raboty.md).
