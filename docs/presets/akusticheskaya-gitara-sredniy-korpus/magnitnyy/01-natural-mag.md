# Natural Mag

**Роль:** основной чистый тон для среднего корпуса с магнитным датчиком.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM → EQ Guitar EQ1 → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate3 | Threshold | 20–30 |
| COMP | Sustain | 25–40 |
| AC Pre | Volume / Tone | 55 / 48–55 |
| AC Pre | Balance | 35–50 |
| AC Pre | EQ Freq | ~300–450 Hz |
| AC Pre | EQ Gain | 46–49 (слегка убрать «коробку» магнита) |
| CAB OM | Low / High Cut | 90–110 Hz / 9–10 kHz |
| EQ1 | 125 / 400 / 800 / 1.6k / 4k | −2 / −2 / +1 / +1 / −1 |
| Room | Mix | 15–25 |

## Варианты

1. Ярче струны: EQ 4k 0…+1, High Cut 11 kHz.
2. Больше «микрофона»: CAB → **Bird**, Tone AC Pre чуть вверх.
3. Без AC Pre: AMP **Dark Twin** Gain низкий, Bright OFF + CAB OM.
