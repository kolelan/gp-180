# Singer Edge (магнит)

**Роль:** лёгкий edge for singer-songwriter: сатурация/чуть OD без «электро-соло».

```
NR Gate3 → PRE Saturate → AMP AC Pre (или Dark Twin Gain низкий)
→ CAB OM → EQ Guitar EQ1 → DLY Tape тихо → RVB Spring → VOL
```

### Gate 3 ([описание](../../../modules/01-nr/gate-3/))

Edge/OD: паузы чуть суше. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **25–40** | порог по паузе без игры |
| **Ratio** | **45–60** | жёсткость глушения |
| **Attack** | **5–15** ≈ ms | открытие под атаку |
| **Release** | **120–250** ≈ ms | хвост ноты / тишина |
| **Hold** | **30–70** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix / High Cut | 25–40 / вниз |
| AC Pre / Twin | Gain-эквивалент | низкий |
| CAB | Cuts | 100 Hz / 8.5 kHz |
| EQ1 | 800 / 4k | +2 / −2 |
| Tape | Mix | 10–18 |
| Spring | Mix | 12–20 |

Альтернатива edge: PRE **Penesas** Gain 0–15 как clean boost + AC Pre. Не ставьте TS с высоким Gain — магнит быстро уйдёт в электро-кранч.
