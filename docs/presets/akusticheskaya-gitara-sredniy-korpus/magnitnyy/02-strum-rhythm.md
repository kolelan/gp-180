# Strum Rhythm (магнит)

**Роль:** ритм-бой рядом с басом, вокалом, ударными — плотно, но не гулко.

```
NR Gate2 → PRE COMP4 → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → DLY off → RVB Room Mix низкий → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate2 | Threshold | 35–50 |
| COMP4 | Sustain / Attack | 40 / быстрый |
| AC Pre | Tone | 42–50 |
| AC Pre | EQ Freq / Gain | ~350 Hz / 44–48 |
| CAB | Low / High Cut | 100–120 Hz / 8 kHz |
| EQ2 | 100 / 500 / 1k / 3k / 6k | −4 / −2 / +1 / −1 / −3 |
| Room | Mix | 8–15 |

Магнит на бое легко «съедает» середину вокала на 300–500 Гц — держите вырез в этой зоне.
