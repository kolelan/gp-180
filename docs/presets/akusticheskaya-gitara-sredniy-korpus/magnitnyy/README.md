# Магнитный датчик (soundhole) — средний корпус, сталь

Магнит у розетки звучит теплее пьезо, но теряет «воздух дерева» и часто даёт густую середину 200–500 Гц. Цель: вернуть прозрачность OM-корпуса, не превратить гитару в полуакустический jazz-box без атаки.

## Правила

1. **AC Pre** или чистый **Dark Twin / J-120** + CAB **OM**.
2. Можно слегка **AC Sim** (Mode Standard / Enhanced): Body умеренно, Top не в край — магнит уже не пьезо.
3. Режьте 200–400 Гц при гуле; поднимайте 2–4 кГц осторожно для блеска струн.
4. Saturate Mix низкий (10–20) — добавить «запись», не кранч.
5. Feedback реже, чем у пьезо, но на мониторе Low Cut всё равно держите.
