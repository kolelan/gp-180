# Акустическая гитара: средний корпус, стальные струны

Для корпусов класса **OM / 000 / Auditorium / Grand Concert** (не dreadnought и не jumbo): более ровный баланс, меньше «гудящего» низа, удобно и под бой, и под фингерстайл.

Отдельно наборы под тип датчика:

| Датчик | Папка | Характер сигнала |
|--------|-------|------------------|
| **Магнитный** (soundhole / магниты у розетки) | [magnitnyy/](./magnitnyy/) | Теплее, «электроподобнее», меньше quack, часто мутноватая середина |
| **Пьезо** (седылка / бридж) | [piezo/](./piezo/) | Атака + «квок», мало дерева, риск обратной связи |

Если на гитаре **два источника (blend)**: начните с пьезо-пресета на 60–70% и магнитного тона как «тепло снизу», либо используйте аппаратный бленд на инструменте и один из Natural-пресетов с лёгкой донастройкой.

## Почему CAB = OM

На GP-180 акустический кабинет **OM** ближе всего к среднему корпусу. `AC` / `JUMBO` — больше «дред»; `Bird` — ярче и «иконичнее». Для среднего корпуса по умолчанию: **OM**, запасной — **Bird**.

## Общее для обоих датчиков

1. Стальные струны дают сильный щелчок медиатора — COMP мягкий, High Cut обязателен в бэнде.
2. HPF (CAB Low Cut) ~80–120 Гц (магнит), ~100–150 Гц (пьезо / сцена).
3. Место в миксе: басу отдать низ ниже 120 Гц, вокалу 2–5 кГц не забивать, тарелкам не конкурировать на 6–10 кГц.
4. База усиления: **AC Pre** + акустический CAB; хайгейн amp почти не нужен.
