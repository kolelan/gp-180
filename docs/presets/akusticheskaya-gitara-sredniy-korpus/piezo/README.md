# Пьезо — средний корпус, сталь

Пьезо на OM/000 обычно: щелчок медиатора, quack ~1–2.5 кГц, мало тела деки, howl на сцене. Средний корпус сам по себе не «гудит» как dreadnought — не раздувайте Bass; восстанавливайте дерево через **AC Pre + CAB OM** и мягкую сатурацию.

## Правила

1. База: **AC Pre** + **CAB OM** (не JUMBO без нужды).
2. Notch/вырез на quack (AC Pre parametric ~1.2–2 kHz, Gain ниже 50).
3. + тело на 200–400 Гц умеренно (+1…+3 в Guitar EQ).
4. High Cut 6–9 кГц; Saturate Mix 15–30 при «пластике».
5. Отдельный anti-feedback пресет для мониторов.
