# Strum Anti-Quack (пьезо)

**Роль:** бой — убрать квок и шип, оставить щелчок сетки.

```
NR Gate2 → PRE COMP4 → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → RVB Room Mix минимальный → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate2 | Threshold | 40–55 |
| COMP4 | Sustain / Attack | 40 / быстрый |
| AC Pre | Tone | 38–46 |
| AC Pre | EQ Freq / Gain | ~1.6–2.2 kHz / 40–46 |
| CAB | Low / High Cut | 120–150 Hz / 7 kHz |
| EQ2 | 100 / 500 / 1k / 3k / 6k | −4 / −1 / −3 / −2 / −5 |
| Room | Mix | 5–12 |

В бэнде с барабанами не поднимайте 6 kHz — тарелки уже там.
