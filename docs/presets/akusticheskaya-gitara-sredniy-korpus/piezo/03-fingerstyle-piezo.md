# Fingerstyle Piezo

**Роль:** перебор: ясные высокие и читаемый бас без howl.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ1 → DLY Slapback тихо → RVB Plate → VOL
```

### Gate 3 ([описание](../../../modules/01-nr/gate-3/))

Мягкий сустейн / clean. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **22–30** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **200–320** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–90** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 25–40 |
| AC Pre | Tone | 48–55 |
| AC Pre | EQ Freq | ~1.5 kHz (quack) или ~250 Hz (гул) |
| AC Pre | EQ Gain | 45–49 |
| CAB | Low / High Cut | 100 Hz / 9 kHz |
| EQ1 | 125 / 400 / 1.6k / 4k | −2 / +2 / −1 / −2 |
| Slapback | Mix | 8–14 |
| Plate | Mix | 12–20 |

Динамику пальцев сохраняйте: не задирайте COMP4 Clipping.
