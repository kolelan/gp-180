# Ambient Piezo

**Роль:** пады и лупы с сохранением акустического характера.

```
NR → PRE COMP → AMP AC Pre → CAB OM → EQ Guitar EQ2
→ MOD Auto Swell → DLY Dual Echo → RVB Sweet Space / Shimmer → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| AC Pre | EQ Gain @ ~1.6 kHz | 46–49 |
| CAB | Cuts | 110 Hz / 8 kHz |
| EQ2 | 6k | −4 |
| Auto Swell | Attack | средний |
| Dual Echo | Mix | умеренный, разные Time A/B |
| Sweet Space | Mix | 20–30 |
| Shimmer на сцене | Mix | ≤12–15 |

На мониторах сначала проверьте [Anti-FB](./05-stage-anti-feedback.md) — длинные хвосты провоцируют howl.
