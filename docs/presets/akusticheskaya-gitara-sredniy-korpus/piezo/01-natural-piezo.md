# Natural Piezo

**Роль:** основной «живой» OM-тон с пьезо.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM → EQ Guitar EQ1 → RVB Room → VOL
```

### Gate 3 ([описание](../../../modules/01-nr/gate-3/))

Мягкий сустейн / clean. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **22–30** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **200–320** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–90** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 30–45 |
| AC Pre | Tone / Balance | 42–52 / 35–50 |
| AC Pre | EQ Freq | ~1.4–1.8 kHz |
| AC Pre | EQ Q | средний |
| AC Pre | EQ Gain | 44–48 (вырез quack) |
| CAB OM | Low / High Cut | 110–130 Hz / 8 kHz |
| EQ1 | 125 / 400 / 800 / 1.6k / 4k | −2 / +2 / 0 / −2 / −3 |
| Room | Mix | 15–25 |

## Варианты

1. Теплее: PRE **Saturate** Mix 20 (вместо сильного +Bass).
2. Ярче дека: CAB **Bird**, High Cut 9 kHz, EQ 4k −2 (не +).
3. Запись «в линию»: CAB OFF + жёстче High Cut в EQ.
