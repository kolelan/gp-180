# Capo Control

**Роль:** каподастр / игра выше — пьезо становится резче; придержать шип и quack.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → DLY Digital тихо → RVB Plate → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 30–45 |
| AC Pre | Tone | 38–46 |
| AC Pre | EQ Freq / Gain | ~1.8–2.5 kHz / 42–47 |
| CAB | Low / High Cut | 120 Hz / 7–8 kHz |
| EQ2 | 1k / 3k / 6k | −1 / −3 / −5 |
| Delay | Mix | 8–14 |
| Plate | Mix | 12–18 |

Без капы тон кажется тёмным — вернитесь к [Natural](./01-natural.md).
