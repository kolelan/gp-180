# Natural Piezo

**Роль:** основной «живой» тон пьезо на стали.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM → EQ Guitar EQ1 → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate3 | Threshold | 20–35 |
| COMP | Sustain | 30–45 |
| AC Pre | Tone / Balance | 42–52 / 35–50 |
| AC Pre | EQ Freq | ~1.4–1.8 kHz |
| AC Pre | EQ Q | средний |
| AC Pre | EQ Gain | 44–48 |
| CAB OM | Low / High Cut | 110–130 Hz / 8 kHz |
| EQ1 | 125 / 400 / 800 / 1.6k / 4k | −2 / +2 / 0 / −2 / −3 |
| Room | Mix | 15–25 |

**Варианты:** теплее → Saturate Mix 20; ярче дека → CAB Bird + High Cut 9 kHz; крупный dread-характер → CAB AC (осторожно с низом).
