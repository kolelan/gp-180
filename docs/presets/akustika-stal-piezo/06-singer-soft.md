# Singer Soft

**Роль:** аккомпанемент под вокал — смягчить атаку пьезо, не маскировать слова.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → DLY Pure Delay тихо → RVB Plate Mix низкий → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 35–50 |
| AC Pre | Tone | 40–48 |
| AC Pre | EQ Freq / Gain | ~2–3 kHz / 45–48 |
| CAB | Low / High Cut | 120 Hz / 7.5 kHz |
| EQ2 | 1k / 3k / 6k | 0 / −2 / −4 |
| EQ2 | 500 | −1…−2 |
| Delay | Mix | 8–12 |
| Plate | Mix | 10–18 |

Конфликт с вокалом на 2–5 кГц — режьте гитару, не «ярчите» голос в мониторе.
