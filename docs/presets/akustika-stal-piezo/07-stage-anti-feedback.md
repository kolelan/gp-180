# Stage Anti-Feedback

**Роль:** громкие мониторы / комбик у деки — ловить howl.

```
NR Gate3 → PRE COMP4 → AMP AC Pre → CAB GA (или OM)
→ EQ Guitar EQ1 → RVB off или Room ≤8 → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Сцена / anti-FB: суше паузы. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **35–50** | порог по паузе без игры |
| **Ratio** | **50–70** | жёсткость глушения |
| **Attack** | **5–15** ≈ ms | открытие под атаку |
| **Release** | **100–200** ≈ ms | хвост ноты / тишина |
| **Hold** | **20–50** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| AC Pre | Tone | 35–45 |
| AC Pre | EQ Freq | крутить **на ноте howl** |
| AC Pre | EQ Q | уже |
| AC Pre | EQ Gain | 35–44 |
| CAB | Low Cut | 140–180 Hz |
| CAB | High Cut | 6–7.5 kHz |
| EQ1 | 125 / 400 | −5…−8 / −2…−5 |
| EQ1 | 1.6k / 4k | −2 / −4 |
| Patch Volume | | ниже Natural (−3…−6 дБ ориентир) |

Алгоритм: к монитору → EQ Freq AC Pre → углубить Gain → добить EQ1.
