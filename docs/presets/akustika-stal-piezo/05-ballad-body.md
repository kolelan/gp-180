# Ballad Body

**Роль:** медленные песни — вернуть тело деки.

```
NR Gate3 → PRE Saturate → AMP AC Pre → CAB OM (или Bird)
→ EQ Guitar EQ1 → DLY Sweet Echo → RVB Hall → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix / High Cut | 20–35 / вниз |
| AC Pre | Tone / Balance | 45–55 / 40–55 |
| AC Pre | EQ Freq / Gain | ~900 Hz–1.2 kHz / 48–52 |
| CAB | Low / High Cut | 100 Hz / 8.5–9 kHz |
| EQ1 | 400 / 800 / 4k | +3 / +1 / −4 |
| Sweet Echo | Mix | 10–18 |
| Hall | Mix / Decay | 22–32 / средний |

Jumbo CAB только если нужен намеренно «большой» низ — на OM/000 часто лишний.
