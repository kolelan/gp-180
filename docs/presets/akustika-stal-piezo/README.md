# Гитара · сталь · пьезо

Чистый набор **только** для акустической гитары с **пьезодатчиком** (седылка / бридж / undersaddle). Без магнита и без бленда.

Пьезо на стали: острая атака медиатора, «квок» ~1–2.5 кГц, мало дерева деки, риск **обратной связи** у мониторов. Цель — вернуть тело корпуса, убрать quack, посадить тон рядом с басом, вокалом и ударными.

## Базовая цепь

```
NR → PRE (COMP / Saturate) → AMP AC Pre → CAB OM / Bird / AC
→ EQ → (MOD/DLY) → RVB короткий на сцене → VOL
```

## Правила

1. **AC Pre** обязателен: parametric EQ — главный инструмент против quack и howl.
2. COMP мягкий (бой) или чуть сильнее (выровнять пики медиатора); не душить фингерстайл.
3. Saturate Mix 15–30 — тепло без электро-овердрайва.
4. CAB Low Cut ~100–150 Гц; High Cut ~6–9 кГц.
5. Тело: +200–400 Гц умеренно; не раздувать Bass на amp.
6. На сцене — отдельный Anti-FB; длинный Shimmer только в спецпатче и тихо.

См. [EQ в миксе](../../eq-mix/). Смешанный набор (магнит+пьезо по папкам): [../akusticheskaya-gitara-sredniy-korpus/piezo/](../akusticheskaya-gitara-sredniy-korpus/piezo/). Только магнит: [../akustika-stal-magnit/](../akustika-stal-magnit/).
