# Warm Hall

**Роль:** тёплое пространство для акустических номеров без пьезо-шипения.

```
NR Gate3 → PRE Saturate → AMP AC Pre → CAB Bird
→ EQ Guitar EQ1 → MOD Detune еле → RVB Hall или Tube Spring → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Длинные ноты / pad. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **15–28** | порог по паузе без игры |
| **Ratio** | **25–45** | жёсткость глушения |
| **Attack** | **10–40** ≈ ms | открытие под атаку |
| **Release** | **400–1200** ≈ ms | хвост ноты / тишина |
| **Hold** | **80–200** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix | 20–30 |
| AC Pre | Tone | 44–52 |
| AC Pre | EQ Freq / Gain | ~1.5 kHz / 45–48 |
| CAB | Cuts | 110 Hz / 8 kHz |
| EQ1 | 400 / 4k | +2 / −4 |
| Detune | Wet / cents | 10–18 / ±5…8 |
| Hall / Spring | Mix | 20–30 |

Без длинного Shimmer на сцене — иначе howl + каша с тарелками.
