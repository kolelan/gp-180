# Flatpick Sparkle

**Роль:** медиаторный фолк/кантри — атака есть, пьезо-шип под контролем.

```
NR Gate3 → PRE COMP4 → AMP AC Pre → CAB Bird
→ EQ Guitar EQ1 → DLY Slapback → RVB Room → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Мягкий сустейн / clean. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **22–30** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **200–320** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–90** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| COMP4 | Sustain / Attack | 40 / быстрый |
| AC Pre | Tone | 45–52 |
| AC Pre | EQ Freq / Gain | ~1.7 kHz / 44–48 |
| CAB Bird | Low / High Cut | 110 Hz / 8–8.5 kHz |
| EQ1 | 800 / 1.6k / 4k | +1 / −2 / −3 |
| Slapback | Mix / Time | 12–18 / 90–120 ms |
| Room | Mix | 10–18 |

Нужно тепло: Saturate Mix 15–25 вместо подъёма Treble.
