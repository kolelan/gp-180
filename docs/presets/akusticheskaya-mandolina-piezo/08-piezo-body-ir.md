# Piezo + Body IR

**Роль:** пьезо-мандолина через **User IR** (лучше свой/мандо-профиль; гитарный dreadnought — только как эксперимент).

Сценарий: [../../ir-acoustic/scenarii/piezo-body-ir-plus-fx.md](../../ir-acoustic/scenarii/piezo-body-ir-plus-fx.md).

```
NR Gate3 → PRE COMP → AMP OFF (или AC Pre мягко) → N→S OFF
→ CAB User IR → EQ Guitar EQ2
→ DLY Slapback тихо → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 25–40 |
| CAB | Low / High Cut | 120 Hz / 7.5–8.5 kHz |
| EQ2 | 1k / 3k / 6k | −2 / −2 / −4 |
| Room | Mix | 12–20 |

Профили: [`../../ir-acoustic/mandolina/`](../../ir-acoustic/mandolina/).
