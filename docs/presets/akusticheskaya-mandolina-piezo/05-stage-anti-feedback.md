# Stage Anti-Feedback

**Роль:** громкие мониторы / акустический комбик рядом с инструментом; меньше howl.

```
NR Gate3 → PRE COMP4 → AMP AC Pre → CAB GA (или AC)
→ EQ Guitar EQ1 (агрессивный HPF-характер полосами)
→ MOD off → DLY off → RVB Room Mix минимальный → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate3 | Threshold / Hold | 35–50 / короткий–средний |
| COMP4 | Sustain | 30–40 |
| AC Pre | Tone | 35–45 |
| AC Pre | EQ Freq | подберите **на слух к ноте howl** (часто 200–400 или 1–2.5 kHz) |
| AC Pre | EQ Q | уже (уже Q = точечный notch) |
| AC Pre | EQ Gain | 35–45 (глубокий вырез) |
| CAB | Low Cut | 140–180 Hz |
| CAB | High Cut | 6–7.5 kHz |
| EQ1 | 125 | −4…−8 |
| EQ1 | 400 | −2…−5 (если гудит низ-середина) |
| EQ1 | 1.6k / 4k | −2 / −4 |
| Room | Mix | 0–10 |
| Patch Volume | | ниже обычного clean (−3…−6 дБ ориентир) |

## Как ловить частоту

1. Включите монитор, подойдите к нему корпусом.
2. Крутите AC Pre **EQ Freq**, пока howl не пропадёт / не ослабнет, затем углубите **EQ Gain**.
3. При необходимости второй «полкой» добейте Guitar EQ1.

## CTRL

Ещё −Volume или глубже notch на самый громкий куплет.
