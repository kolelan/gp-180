# Bluegrass Chop (пьезо)

**Роль:** ритм-чоп, backbeat — максимум атаки, минимум quack и реверба.

```
NR Gate2 → PRE COMP4 → AMP AC Pre → CAB AC
→ EQ Guitar EQ2 → MOD off → DLY off → RVB Room Mix минимальный → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate2 | Threshold / Attack | 40–55 / короткий |
| COMP4 | Sustain / Attack | 35–45 / быстрый |
| AC Pre | Tone | 35–45 |
| AC Pre | EQ Freq | ~1.5–2.0 kHz |
| AC Pre | EQ Gain | 40–46 (заметный вырез quack) |
| CAB AC | Low Cut / High Cut | 130 Hz / 7 kHz |
| EQ2 | 100 / 500 / 1k / 3k / 6k | −4 / −1 / −2 / −2 / −5 |
| Room | Mix | 5–10 |

## Зачем так

Пьезо на чопе «щёлкает» в 1–3 кГц и орёт на 6 кГц; Gate помогает паузам между ударами; реверб почти выключен, чтобы не смазать грид с банджо/аку-гитарой.

## CTRL

Чуть поднять Volume / отпустить Gate для fill-линий.
