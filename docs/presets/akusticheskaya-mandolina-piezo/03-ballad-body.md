# Ballad Body

**Роль:** медленные песни, тремоло, тёплое тело корпуса.

```
NR Gate3 → PRE COMP → PRE не дублировать: AMP AC Pre
→ CAB JUMBO (или Bird) → EQ Guitar EQ1
→ DLY Slapback очень тихо → RVB Hall или Plate → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 30–45 |
| AC Pre | Tone / Balance | 45–55 / 40–55 |
| AC Pre | EQ Freq | ~800 Hz–1.2 kHz |
| AC Pre | EQ Gain | 50–54 (лёгкий подъём тела, не quack) |
| CAB JUMBO | Low Cut / High Cut | 100 Hz / 9 kHz |
| EQ1 | 125 / 400 / 800 / 1.6k / 4k | −1 / +3 / +1 / −1 / −4 |
| Slapback | Mix / Time | 8–15 / 90–120 ms |
| Hall | Mix / Pre Delay / Decay | 20–30 / чуть / средний |

## Вариант «ещё теплее»

Перед AMP: **Saturate** Mix 20–30, High Cut вниз. Не поднимайте Bass на amp сильно — лучше 400 Hz в EQ.

## Микс

Дайте басу и бочке низ; ваш +400 Hz не должен стать «коробкой» — если басист жалуется, опустите 400 и поднимите 800 умеренно.
