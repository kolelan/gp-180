# Мандолина · пьезо

Пьезо на акустической мандолине обычно даёт **узкий «квок»**, резкую атаку медиатора, мало «дерева» корпуса и склонность к **обратной связи** на сцене с мониторами. Задача цепи — вернуть тело, убрать quack, не раздуть сибилянты и оставить место басу / ритму / вокалу / тарелкам.

Отличие от [мандолины · магнит (мандокастер)](../mandocaster/): там борются с ярким магнитным ВЧ; здесь — с пьезо-квоком и обратной связью, плюс чаще используют **AC Pre** и акустические CAB/IR.

## Общие правила

1. **AC Pre** (AMP) или мягкий clean amp + акустический CAB (`Bird` / `OM` / `AC`) — база «как DI акустики».
2. **COMP** мягкий: выровнять пики медиатора, не душить тремоло.
3. **Saturate** в PRE (Mix 15–30) — дешёвый способ добавить тепло и склеить «пластиковый» пьезо.
4. EQ: типичный quack ~**1–2.5 кГц** резать; тело ~**200–400 Гц** поднимать осторожно (+1…+3); воздух не через +6 кГц, а через контролируемый High Cut.
5. **CAB Low Cut** ~100–150 Гц (обратная связь и гул); **High Cut** ~6–9 кГц.
6. На сцене: короткий Room, минимум Shimmer; отдельный пресет **Anti-Feedback**.
7. N→S/хайгейн amp — редко; если SnapTone, то только open clean (Dark CL / Dark PRI) + сильный High Cut.

См. также: [EQ в миксе](../../eq-mix/), модули [AC Pre](../../modules/06-amp/ac-pre/), [AC Sim](../../modules/02-pre/ac-sim/), [Saturate](../../modules/02-pre/saturate/).
