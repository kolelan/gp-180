# Celtic Lead

**Инструмент:** мандокастер (мандолина · магнит).  
**Стиль:** кельт / фолк-лид, орнаменты, арпеджио, мелодии поверх бэнда.  
**Характер:** чистый «колокольчик» (chime), воздух и ширина — без рок-кранча и без жёсткого пьезо-шипа.

Связанные термины: [Slapback](../../terminy/slapback.md) тут **не** главный эффект (здесь ритмический Digital Delay); [FRFR](../../terminy/frfr.md); [HPF](../../terminy/hpf.md).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Мелодия, трели, орнаменты | Ритм-чоп → [Bluegrass Chop](./02-bluegrass-chop.md) |
| Яркий AC30-like «воздух» | Тёплый балладный clean → [Folk Clean](./01-folk-clean.md); лид с grit → [Bright Lead](./06-bright-lead.md) |
| Лид слышен рядом с аку-гитарой и вокалом | Стена delay/shimmer → [Ambient Pad](./05-ambient-pad.md) |

Идея: британский **chime** (Foxy / VOX-like) + лёгкий **chorus** (ширина) + **delay в темп** (1/8) + короткий **Plate** — «кельтский» лид, а не баллада Fender Twin.

---

## Цепь

```
NR Gate3
→ PRE COMP
→ N→S Foxy CL          (или AMP Foxy 30N / Match CL)
→ CAB Foxy 2x12        (OFF, если SnapTone уже с кабинетом)
→ EQ Guitar EQ2
→ MOD G-Chorus
→ DLY Digital          (Sync ON, доля 1/8)
→ RVB Plate
→ VOL
```

На [FRFR](../../terminy/frfr.md)/наушниках CAB обычно ON. В INPUT гитарного комбика CAB часто OFF.

### Зачем каждый блок

| Модуль | Роль здесь |
|--------|------------|
| **Gate3** | Тихие паузы между фразами без обрезания сустейна орнаментов (порог мягче, чем у чопа) |
| **COMP** | Выровнять атаку медиатора, чтобы ornament'ы не «прыгали» по громкости |
| **Foxy CL** | Открытый британский clean/chime; Gain низкий — магнит мандолины и так яркий |
| **Foxy 2x12** | Подчёркивает «колокольчик»; не берите V30-scoop → [почему](../../ir-acoustic/terminy-mic-ir-v30-scoop.md) |
| **Guitar EQ2** | Контроль шипа на **6 kHz** (мандокастер легко орёт) |
| **G-Chorus** | Лёгкая стереоширина / «воздух»; Depth умеренный, иначе размажет атаку |
| **Digital Delay** | Чёткие повторы в грид (кельтские фразы любят ритмичный echo), не размытый [Slapback](../../terminy/slapback.md) |
| **Plate** | Короткий студийный хвост; не Hall/Shimmer на плотной сцене |

Альтернатива без SnapTone: **AMP Foxy 30N** или **Foxy 30TB** Char Cool + CAB Foxy 2x12.

## Стартовые параметры

| Блок | Параметр | Значение | Комментарий |
|------|----------|----------|-------------|
| Gate3 | Threshold | 20–35 | Ниже, чем у chop-пресета |
| COMP | Sustain | 25–40 | Не душить динамику пальцев/медиатора |
| N→S / AMP | Gain | **~20** | Chime, не crunch |
| | Treble | **~35** | Магнит уже яркий |
| | Presence | **~40** | Не в потолок |
| | Bass | 40–50 | + [Low Cut](../../terminy/low-cut-high-cut.md) ~100 Hz |
| CAB | High Cut | ~8–9 kHz | Снять «песок» |
| EQ2 | 100 Hz | −3…−4 | Место басу/бочке |
| EQ2 | 1 kHz | +1…+2 | Читаемость мелодии |
| EQ2 | **6 kHz** | **−3** | Главный антишип |
| G-Chorus | Depth | **~25** | Rate средний; Sync по вкусу |
| Digital | Mix | **~20** | |
| Digital | Time | **1/8**, Sync **ON** | Под BPM патча / TAP |
| Digital | Feedback | низкий–средний | 1–2 слышимых повтора, не лента |
| Plate | Mix | **~18** | Decay короткий–средний |

Если лид «тонет» в бэнде — чуть **+1 kHz** / Mid, не +Treble.  
Если режет ухо — ещё −6 kHz и High Cut вниз.

## Параметры эффектов — что крутить и зачем

Ниже только те ручки, которые **реально важны** для Celtic Lead. Остальное можно оставить по умолчанию / нейтрально.

### NR — Gate 3

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Threshold** | Ниже порога — тишина | **20–35**. Выше — обрежет тихие орнаменты |
| **Attack** | Как быстро «открывается» | Короткий–средний |
| **Release** | Как быстро глушит шум | Средний |
| **Hold** | Держит гейт открытым | Небольшой, чтобы не рубить сустейн |

Карточка: [Gate 3](../../modules/01-nr/gate-3/).

---

### PRE — COMP

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Sustain** | Сила сжатия | **25–40**. Больше — «размажет» динамику орнаментов |
| **Volume** | Выход компрессии | Чуть выше/около unity, чтобы после COMP уровень не просел |

Карточка: [COMP](../../modules/02-pre/comp/).

---

### N→S — Foxy CL (или AMP Foxy)

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Gain** | Насыщение преампа | **~20** — chime, не crunch |
| **Bass** | Низ усилка | **40–50** + Low Cut в CAB |
| **Middle** | Середина | ~50–55 |
| **Treble** | Верх усилка | **~35** — магнит уже яркий |
| **Presence** | «Воздух» / атака верха | **~40**, не максимум |
| **VOL** | Громкость после модели | Под патч / сцену |

Если AMP **Foxy 30N**: смотрите **Tone cut** (против часовой = темнее) и **Bright** — для Celtic чаще Bright OFF или слабый.

---

### CAB — Foxy 2x12 (или OFF)

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Volume** | Уровень кабинета | Под общий баланс патча |
| **Low Cut** | [HPF](../../terminy/hpf.md) — режет низ | **~100 Hz** |
| **High Cut** | Режет самый верх | **~8–9 kHz** |
| **Precision** | 1024 / 2048 точек | Regular ок; High — если DSP позволяет |

Смысл Low/High Cut: [термин](../../terminy/low-cut-high-cut.md).

---

### EQ — Guitar EQ 2

Полосы: **100 / 500 / 1k / 3k / 6k** Гц + Volume.

| Полоса | Частота | Для Celtic Lead |
|--------|---------|-----------------|
| Band 1 | 100 Hz | **−3…−4** — место басу/бочке |
| Band 2 | 500 Hz | около 0 / чуть −1 |
| Band 3 | 1 kHz | **+1…+2** — читаемость мелодии |
| Band 4 | 3 kHz | 0…−1 (не спорить с вокалом) |
| Band 5 | **6 kHz** | **−3** — главный антишип мандокастера |
| Volume | выход EQ | нейтраль / чуть подкрутить соло |

Карточка: [Guitar EQ 2](../../modules/08-eq/guitar-eq-2/).

---

### MOD — G-Chorus

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Depth** | Глубина «волны» | **~25**. Больше — каша в быстрых орнаментах |
| **Rate** | Скорость модуляции | средний |
| **Volume** | Уровень эффекта | не громче сухого сигнала |
| **Sync** | К TAP/BPM | по вкусу; для лида часто OFF (свободный rate) |

Карточка: [G-Chorus](../../modules/09-mod/g-chorus/).

---

### DLY — Digital Delay

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Mix** | Мокро/сухо | **~20** (на соло CTRL ↑) |
| **Feedback** | Число повторов | низкий–средний (1–2 слышимых) |
| **Time** | Задержка | доля **1/8** при Sync ON |
| **Time R%** | Правый канал vs левый | ~100% или чуть иначе для ширины |
| **Spread** | Стереоширина | умеренно |
| **Level** | Выход delay | около нормы |
| **Sync** | К темпу патча | **ON** |
| **Trail** | Хвост при выключении модуля | ON удобно для CTRL |

Это **delay**, не reverb и не [Slapback](../../terminy/slapback.md) (Slapback — короткий «шлёпок» 80–120 ms без привязки к 1/8).

Карточка: [Digital Delay](../../modules/10-dly/digital-delay/).

---

### RVB — Plate

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Mix** | Мокро/сухо | **~18** (на сцене ниже) |
| **Decay** | Длина хвоста | короткий–средний |
| **High Damp** | Приглушение верха в хвосте | чуть вверх = меньше «шипа» в reverb |
| **Trail** | Хвост при байпасе | по вкусу |

Карточка: [Plate](../../modules/11-rvb/plate/).

---

### VOL

| Параметр | Смысл | Для Celtic Lead |
|----------|--------|-----------------|
| **Volume** | Чистый уровень | Для свэллов на EXP — поставьте VOL **до** Delay/Reverb |

Карточка: [Volume](../../modules/12-vol/volume/).

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **Gain** (Foxy) — chime vs грязь  
2. **EQ 6 kHz** — резкость  
3. **Chorus Depth** — ширина  
4. **Delay Mix** + **Sync/Time 1/8** — ритмический воздух  
5. **Plate Mix** — хвост  
6. **CAB Low Cut** — гул vs бас

---

## Игра и управление

### CTRL (рекомендации)

- **Delay Mix ↑** только на открытое соло / длинную фразу.  
- Опционально: Chorus OFF в куплет (суше рядом с вокалом), ON в проигрыш.

### EXP

- Для «скрипичных» свэллов: модуль **VOL** поставить **раньше Delay/Reverb** в цепи и назначить на EXP — нарастание громкости, а хвосты delay остаются.  
- Не крутите Wah без нужды: для кельт-лида обычно лишнее.

### Правая рука

Тремоло и орнаменты любят ровную динамику — поэтому COMP умеренный. Резкий чоп этому патчу противопоказан (уйдёт в шип).

## Посадка в ансамбль

Рядом с **басом · аку-ритмом · электро · ударными · вокалом**:

- Мандо-лид занимает ~**1.5–3 кГц**; не дублируйте тот же пик ярким электро.  
- [Низ](../../terminy/niz-v-mikse.md) на струнном режьте ([HPF](../../terminy/hpf.md)).  
- Chorus + delay + plate вместе легко «зальют» тарелки — на сцене Mix'ы ниже, чем в наушниках.

См. микс-фолк: [../elektro-ansambl/mix-folk.md](../elektro-ansambl/mix-folk.md).

---

## Варианты

1. **Ещё «аккуратнее» в миксе:** Chorus OFF, Delay Mix 12–15, Plate 12.  
2. **Больший chime:** Foxy CL → Match 35 CL / Foxy 30TB Cool; High Cut не поднимайте.  
3. **Без SnapTone:** AMP Foxy 30N + CAB Foxy 2x12.  
4. **Наушники / луп:** можно чуть поднять Plate; на мониторах верните вниз (howl / каша).

---

## Когда пресет не подходит

- Нужен **ритм-чоп** — патч слишком «мокрый»; берите [Bluegrass Chop](./02-bluegrass-chop.md).  
- Нужен лид **с лёгкой грязью**, но без «стекла» сверху — [Bright Lead](./06-bright-lead.md).  
- Нужен **тёмный dark-americana** без chime — скорее [Folk Clean](./01-folk-clean.md); электро-эстетика — [dark clean в ансамбле](../elektro-ansambl/05-dark-americana-clean.md).  
- **Громкие мониторы с howl** — сначала логика [Anti-FB](../akusticheskaya-mandolina-piezo/05-stage-anti-feedback.md) / жёстче Low Cut, потом уже лид.
