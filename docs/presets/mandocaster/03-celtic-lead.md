# Celtic Lead

**Роль:** мелодии, орнаменты, «воздух».

```
NR Gate3 → PRE COMP → N→S Foxy CL → CAB Foxy 2x12 (или OFF если cab в ST)
→ EQ Guitar EQ2 → MOD G-Chorus → DLY Digital → RVB Plate → VOL
```

| Параметр | Значение |
|----------|----------|
| Gain | 20 |
| Treble / Presence | 35 / 40 |
| Chorus Depth / Rate | 25 / средний |
| Delay Mix / Time | 20 / 1/8 ноты Sync ON |
| Plate Mix | 18 |
| EQ 6 kHz | −3 |

**CTRL:** поднять Delay Mix на соло.
**EXP:** Volume перед Delay (сдвиньте VOL раньше в цепи) для скрипичных свэллов.
