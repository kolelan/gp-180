# Folk Clean

**Инструмент:** мандокастер (магнит).  
**Стиль:** песни, аккомпанемент вокала, спокойные мелодии.  
**Характер:** тёплый american clean (**AMP Dark Twin** / Twin-like), короткие хвосты — без кельт-chime и без чопа.

См. также: [Celtic Lead](./03-celtic-lead.md) (ярче/шире), [Bluegrass Chop](./02-bluegrass-chop.md) (ритм), amp [Dark Twin](../../modules/06-amp/dark-twin/).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Основной «дневной» clean у вокала | Кельт-chime → [Celtic Lead](./03-celtic-lead.md) |
| Тёплый Twin без [шипа](../../terminy/ship.md) | Чоп → [Bluegrass Chop](./02-bluegrass-chop.md) |
| Лёгкий слэп пространств | Длинный ambient → [Ambient Pad](./05-ambient-pad.md) |

Идея: **COMP** выравнивает медиатор → **Dark Twin** даёт тёплое тело → **EQ** сажает в микс → короткий **Slapback + Room**, не стена.

---

## Цепь

```
NR Gate3 → PRE COMP → AMP Dark Twin → CAB Twin 2x12
→ EQ Guitar EQ1 → DLY Slapback → RVB Room → VOL
```

Альтернатива без AMP: **N→S Dark CL** (SnapTone) + CAB по правилам модели (если cab уже в N→S — CAB OFF). Ручки другие (часто есть **Presence**, нет **Bright**) — см. [карточку Dark Twin](../../modules/06-amp/dark-twin/).

На [FRFR](../../terminy/frfr.md)/наушниках CAB ON. В INPUT гитарного комбика — часто CAB OFF.

---

## Параметры эффектов — что крутить и зачем

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Шумодав **Inverse Expander** в начале цепи. Полные единицы и примеры — в карточке.

Для Folk Clean гейт **мягкий**: важнее сустейн и тихие ноты у вокала, чем идеально мёртвая тишина. Threshold крутите по своей паузе (шум Input / BT / комната).

| Параметр | Диапазон UI | Ориентир Folk Clean | Зачем |
|----------|-------------|---------------------|--------|
| **Threshold** | 0–100 | **22–28** (старт **25**) | Выше — суше паузы, но рискуете отрезать хвосты. Ниже — живее сустейн, чуть больше шороха в паузе |
| **Ratio** | 0–100 | **40–50** (старт **45**) | Мягкое расширение вниз; выше 60 уже ближе к «чопу» — для этого патча рано |
| **Attack** | 1–500 (≈ **ms**) | **8–15** (старт **10**) | Быстро открыть атаку медиатора; больше 30 — щипок тускнеет |
| **Release** | 10–10000 (≈ **ms**) | **200–320** (старт **250**) | Хвост clean-фразы не обрывать; короче 120 — «дырявые» окончания нот |
| **Hold** | 0–1000 (≈ **ms**) | **50–90** (старт **70**) | Тремоло и неоднородная динамика не дёргают гейт |

**Если:** пауза шипит → Threshold +3…5 или Ratio +5.  
**Если:** концы фраз / тремоло рубит → Threshold −3…5, Release +50…100, Hold +20…40.

Чоп / регги-паузы — не этот пресет: [Bluegrass Chop](./02-bluegrass-chop.md), [Reggae Skank](./09-reggae-skank.md).

### COMP
| Параметр | Для Folk Clean |
|----------|----------------|
| **Sustain** | **~30** |
| **Volume** | **~50–55** — **unity**: выход COMP ≈ громкость с выключенным COMP (не бустить «чтобы громче», только компенсировать просадку от сжатия) |

### AMP Dark Twin ([описание](../../modules/06-amp/dark-twin/))

База пресета: **Fender® ’65 Twin Reverb®\***-like clean. Мандокастер уже яркий — **Bright OFF**, Treble не разгонять.

| Параметр | Диапазон / тип | Ориентир Folk Clean | Зачем |
|----------|----------------|---------------------|--------|
| **Gain** | 0–100 (pre) | **20–30** (старт **25**) | Чистота тона. Выше ~35 — уже edge; для «дневного» folk рано. Мало громкости лечите **Volume**, не Gain |
| **Volume** | 0–100 (post) | **50–60** (старт **55**) | Выход amp в цепь / сцена. Сверьте уровень с бэндом |
| **Bass** | 0–100 | **40–50** (старт **45**) | Тело без гула; дальше подчистит CAB Low Cut |
| **Middle** | 0–100 | **52–58** (старт **55**) | Читаемость рядом с вокалом — лучше сюда, чем в Treble |
| **Treble** | 0–100 | **35–42** (старт **40**) | Выше — [шип](../../terminy/ship.md) магнита; яркость лучше EQ **1.6 kHz** |
| **Bright** | ON / OFF | **OFF** | Доп. «стекло» Twin; на мандо почти всегда лишнее |

**Если:** тон глухой на тёмных мониторах → Middle +3…5 или EQ 1.6k, **не** Bright ON в первую очередь.  
**Если:** режет ухо → Treble −5, Bright точно OFF, EQ 4k ниже.  
**Если:** мало жизни в атаке → Gain +3…5 до лёгкого edge, снова проверьте Treble.

Вариант **N→S Dark CL**: Gain ~25; Bass/Mid/Treble примерно как выше; **Presence ~30–40** (если есть); Bright нет. Не переносите Presence в Dark Twin.

### CAB Twin 2x12
| Параметр | Для Folk Clean |
|----------|----------------|
| Low Cut | ~90–110 Hz ([HPF](../../terminy/hpf.md)) |
| High Cut | ~9–10 kHz |
| Volume | под патч (часто ~50–60) |

### Guitar EQ 1 (125 / 400 / 800 / 1.6k / 4k)
| Полоса | Значение | Зачем |
|--------|----------|--------|
| 125 | **−3** | место басу/бочке |
| 400 | **−2** | меньше «коробки» |
| 800 | **+1** | тело |
| 1.6k | **+2** | читаемость |
| 4k | **−3** | антишип магнита |

### DLY Slapback ([описание](../../modules/10-dly/slapback/))

Короткий **delay** («шлёпок»), **не** reverb — [термин](../../terminy/slapback.md). В цепи **до** Room: сначала чёткое эхо, потом комната.

На экране хвост при байпасе = **Trail** (то же, что «Tail»).

| Параметр | Единицы | Ориентир Folk Clean | Зачем |
|----------|---------|---------------------|--------|
| **Mix** | 0–100 | **12–18** (старт **15**) | Доля эха. Выше 25 — слэп спорит с вокалом; ниже 10 — почти не слышно |
| **Time** | **ms** | **90–110** (старт **100**) | Задержка до повтора. ~80 = плотнее; ~120 = чуть «дышнее» |
| **Feedback** | 0–100 | **8–18** (старт **12**) | **Низкий** = один шлёпок. Это не «мало ms», а мало *повторов*: **0–15** ≈ один раз; **>30** уже лента echo |
| **Trail** | ON / OFF | **ON** | При выключении DLY / смене патча повтор доигрывает. **OFF** — обрез хвоста сразу |

**Если:** эхо «учит» вокал → Mix −3…5.  
**Если:** слышна очередь повторов → Feedback вниз к **10**.  
**Если:** при смене патча хлопок обрывается резко → Trail **ON**.

Комнату (пространство) крутите в **RVB Room**, не Feedback.

### RVB Room ([описание](../../modules/11-rvb/room/))

**Reverb** «малая комната» — мягкое пространство **после** Slapback. Slapback = чёткое эхо; Room = размытый хвост (не дублируйте громкость обоих).

На UI хвост при байпасе = **Trail** (то же, что «Tail»).

| Параметр | Единицы | Ориентир Folk Clean | Зачем |
|----------|---------|---------------------|--------|
| **Mix** | 0–100 | **15–22** (старт **18**) | Доля reverb. Выше 28 — «мыло» с вокалом; ниже 12 — сухо, только slap |
| **Pre Delay** | **ms** | **10–25** (старт **15**) | Зазор до reverb: нота + медиатор читаются, потом комната. 0–5 ms — «липкий» хвост; >40 ms — отстранённо |
| **Decay** | **0–100** | **18–28** (старт **22**) | **Короткий** = низкая цифра на шкале: хвост быстро гаснет. **>35** уже не folk-комната; **>50** — риск каши в бэнде |
| **Trail** | ON / OFF | **ON** | При выключении RVB / смене патча хвост доигрывает |

**Если:** мутно с вокалом → Mix −3…5 или Decay −5.  
**Если:** слишком сухо после slap → Mix +3 или Decay +3…5, **не** Feedback на Slapback.  
**Если:** атака «в reverb» → Pre Delay +5…10 ms.

**CTRL (припев):** Room → **Hall** с Mix **18–25**, Decay **короткий–средний** (~25–35), Pre Delay как у Room — чуть больше «воздуха» без длинного shimmer.

### VOL
Чистый уровень патча; при свэллах — VOL **до** delay.

## Три крутилки Quick Access (D / E / F)

На GP-180 три **Quick Access** можно привязать к **любому** параметру патча (Patch Settings / Edit). Ниже — набор именно для **посадки Folk Clean в микс** (сцена / репетиция / запись с вокалом и бэндом): громкость → пространство → тон.

Назначайте в Patch Settings: User Parameter / Quick Access → параметр эффекта.

### Рекомендуемый набор «вход в микс»

| Крутилка | Назначить | Диапазон работы на сцене | Зачем в миксе |
|----------|-----------|--------------------------|---------------|
| **D** | **VOL → Volume** (или Patch Volume) | вокруг стартового уровня ±5…10 | Главный «фейдер» мандолины: сесть рядом с вокалом/аку, не меняя Gain amp (Gain = характер, не баланс) |
| **E** | **RVB Room → Mix** | **12–28** | Сколько «комнаты» в бэнде. Плотнее микс / мониторы → вниз; пусто / вступление → чуть вверх |
| **F** | **EQ Guitar EQ1 → 4 kHz** | **−5…0** (от старта **−3**) | Быстрый [антишип](../../terminy/ship.md): яркие мониторы/хэты → ниже; «пропала» в каше → не +4k, а крутилка **альтернатива** ниже |

Стартовые значения патча не меняйте «в ноль»: крутилки = **живой trim** вокруг уже настроенного Folk Clean.

### Как пользоваться на саундчеке

1. **D (VOL)** — пока мандо не станет «рядом», не громче вокала в куплете.  
2. **E (Room Mix)** — пока хвост не мешает словам и не заливает хэты.  
3. **F (EQ 4k)** — если режет ухо или «ц!» магнита; если тон глухой — см. альтернативу, не крутите 4k в плюс.

Slapback обычно **не** на крутилке: он короткий и стабильный; менять Mix delay на лету рядом с вокалом легко перегрузить. Пространство — через **Room Mix**.

### Альтернативные назначения (если микс другой)

| Ситуация | Вместо F (или вместо E) | Назначить |
|----------|-------------------------|-----------|
| Мандо **тонет**, вокал не режет | вместо 4k | **EQ → 1.6 kHz** (читаемость середины) |
| Нужен «воздух» припева без смены Hall | вместо / доп. к Room | **DLY Slapback → Mix** (**10–20**) |
| Гул с басом/бочкой на площадке | вместо 4k | **CAB → Low Cut** (выше = жёстче обрез низа) |
| Часто переключаете патчи и ловите скачок громкости | оставить D = VOL | не трогать amp Volume крутилкой — только Patch/VOL |

### CTRL (футсвитч) — отдельно от крутилок

Крутилки D/E/F ≠ футсвитч **CTRL**.  
Полные раскладки A/B/C — в разделе **CTRL (Stomp)** ниже. Уровень и «насколько мокро» — крутилки **D/E** и/или EXP.

## CTRL (Stomp): вкл/выкл эффектов ногами

На GP-180 в **Stomp Mode** футсвитчи **A / B / C** включают и выключают назначенные модули ([редактирование](../../manual-ru/06-redaktirovanie-patcha.md), [экраны](../../manual-ru/05-ekrany.md)):

- до **двух модулей на одну кнопку**;
- один модуль — только на **один** футсвитч;
- **C** по умолчанию = CTRL (удержание C → TAP).

Назначение: Edit в Stomp → у модуля FS A / FS B / FS C.

Цепь Folk Clean: `NR → PRE(COMP) → AMP → CAB → EQ → DLY → RVB → VOL`.  
База песни: NR, AMP, CAB, EQ обычно **всегда ON**. На CTRL — то, что меняется по куплетам.

LED: зелёный/красный = суммарное состояние модулей на этой кнопке.

---

### Вариант 1 (рекомендуемый) — «куплет сухой / припев с воздухом»

| Кнопка | Модули | Состояние по умолчанию | Нажатие |
|--------|--------|------------------------|---------|
| **A** | **DLY** (Slapback) | OFF в куплете у вокала | ON на fills / концы фраз |
| **B** | **RVB** (Room) | OFF или едва нужен | ON на припев / проигрыш |
| **C** | — (оставьте под **TAP**) | — | темп, если delay/drum sync |

Идея: два независимых «слоя» пространства. Не дублируйте DLY/RVB на C — модуль может висеть только на одной кнопке.

**Старт сетки:** A OFF, B OFF → сухой аккомпанемент; B ON → комната; A+B ON → слеп + комната.

---

### Вариант 2 — «одна кнопка = весь wet»

| Кнопка | Модули | По умолчанию | Нажатие |
|--------|--------|--------------|---------|
| **A** | **DLY + RVB** | оба OFF | оба ON — «мокрый» припев / outro одним ударом |
| **B** | **PRE** (COMP) | ON | OFF — более открытая динамика / агрессивный щипок |
| **C** | TAP или пусто | — | |

Идея: минимум мышления на сцене. Куплет — сухо; припев — A.  
Минус: нельзя включить только Room без Slapback (они спарены).

---

### Вариант 3 — «сцена с вокалом + запас по компрессии»

| Кнопка | Модули | По умолчанию | Нажатие |
|--------|--------|--------------|---------|
| **A** | **DLY** | OFF | ON на короткие fills |
| **B** | **RVB** | ON (Mix уже низкий ~18) | OFF если мониторы/каша |
| **C** | **PRE** (COMP) | ON | OFF на тихие арпеджио «без клея» |

Идея: B = аварийно убрать комнату; A = точечный слеп; C = характер атаки.

---

### Вариант 4 — «проигрыш громче» (буст)

В слоте **PRE** у Folk Clean уже **COMP** — его лучше оставить всегда ON. Чистый буст для CTRL поставьте в **DST**: эффект **Micro Boost**, по умолчанию **OFF**.

**Почему именно Micro Boost:** поднимает громкость **без** смены характера Twin и **без** лишнего верха. Мандокастер и так яркий — EP-Boost / 14 Boost / Bright легко дают [шип](../../terminy/ship.md).

| Кнопка | Модули | По умолчанию | Нажатие |
|--------|--------|--------------|---------|
| **A** | **DST** (Micro Boost) | OFF | ON на проигрыш |
| **B** | **DLY + RVB** | OFF или только Room ON | ON — пространство на тот же кусок |
| **C** | TAP | — | |

Либо **A = Micro Boost**, **B = DLY**, **C = RVB** — буст и пространство раздельно.

#### Micro Boost — старт

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Gain** | **35–50** (старт **40–45**) | Заметный выход в микс (+~проигрыш), без орёва. Выше 55 — легко в edge на Twin Gain 25 |

Карточка: [Micro Boost PRE](../../modules/02-pre/micro-boost/) / [Micro Boost DST](../../modules/04-dst/micro-boost-dst/).

#### Если хочется чуть другой характер

| Эффект | Когда | Не для Folk Clean, если… |
|--------|--------|---------------------------|
| **Micro Boost** | **выбор по умолчанию** | — |
| **OD 9** Gain **20–30**, Tone **40–48** | нужен mid-push «сквозь» бэнд, не просто громче | уже уходите в grit → ближе [Rock](./04-rock-crossover.md) |
| **B-Boost** Gain низкий, Treble ↓ | чуть «сливочнее» | Treble вверх = шип |
| **Boost** (EP-style) | Bright **OFF**, +3dB по вкусу | Bright ON — стекло на мандо |
| **14 Boost** | почти не | tight/агрессивный, не folk-clean |
| **Penesas** Gain ≈0 | clean boost «klon-style» | Gain выше нуля = уже OD |

Цепь с бустом:

```
NR → PRE COMP → DST Micro Boost (CTRL A) → AMP Dark Twin → …
```

---

### Вариант 5 — «тихое вступление / полный clean»

| Кнопка | Модули | По умолчанию | Нажатие |
|--------|--------|--------------|---------|
| **A** | **DLY + RVB** | ON (мягкий intro) | OFF — сухой вход вокала |
| **B** | **NR** (Gate3) | ON | OFF на свободные хвосты / swell с EXP |
| **C** | TAP | — | |

Редко нужно выключать Gate; имеет смысл только с **EXP Volume swell**, когда гейт кусает тихие заходы.

---

### Что обычно не вешать на CTRL в Folk Clean

| Модуль | Почему |
|--------|--------|
| **AMP / CAB** | Ломает основу тона mid-песней |
| **EQ** | Тон лучше крутилками F / EXP, не ON/OFF |
| **VOL** | Громкость — EXP или Quick Access D, не байпас |

NR почти всегда ON; выключать — только осознанно (вариант 5).

---

### Сочетание CTRL + Quick Access + EXP

| | Live |
|--|------|
| **CTRL** | вкл/выкл Slapback / Room / Boost |
| **D/E/F** | trim Volume, Room Mix, EQ 4k |
| **EXP** | swell (VOL до delay) или Room Mix |

Пример вечера: **вариант 1** + EXP = Volume swell + крутилки «вход в микс».

---

### Как назначить (кратко)

1. Режим **Stomp** (удержание B / MODE).  
2. Edit → модуль → правый Quick Access: **FS A / B / C**.  
3. На одну кнопку — второй модуль тем же FS.  
4. Сохранить патч (Auto Save или SAVE).

## Педаль экспрессии (EXP)

Сначала: **GLOBAL → EXP Calibrate**, затем в патче назначьте параметр на EXP (Edit / Patch Settings / Suite). Jack EXP/FS: тип **EXP** — см. [глобальные настройки](../../manual-ru/07-globalnye-nastroyki.md).

Для Folk Clean Wah / Whammy обычно **не** нужны: clean у вокала, не эффектные «вау».

### Вариант A (основной) — Volume swell

| | |
|--|--|
| **Назначить** | модуль **VOL → Volume** ([карточка](../../modules/12-vol/volume/)) |
| **Положение VOL в цепи** | **до** DLY/RVB (переставьте блок VOL раньше Slapback/Room) |
| **Heel (пятка)** | тихо / почти mute (~0–20) |
| **Toe (носок)** | рабочий уровень патча (~50–60, как стартовый VOL) |

**Зачем:** скрипичные / балладные заходы; хвосты Slapback и Room **остаются**, когда убираете громкость.  
Если VOL стоит в конце цепи (после Room) — при отпускании гасятся и хвосты: для folk swell хуже.

Быстрый ножной «фейдер» в миксе без рук на крутилке D.

### Вариант B — пространство (Room Mix)

| | |
|--|--|
| **Назначить** | **RVB Room → Mix** |
| **Heel** | сухо (**~8–12**) — куплет у вокала |
| **Toe** | чуть воздуха (**~20–28**) — проигрыш / припев |

**Зачем:** одной ногой сушить/смачивать без смены Hall. Не поднимайте toe выше ~30 — каша с вокалом.

Дополняет крутилку E: EXP = live, E = trim на саундчеке.

### Вариант C — лёгкий «кантри-шлёпок» (Slapback Mix)

| | |
|--|--|
| **Назначить** | **DLY Slapback → Mix** |
| **Heel** | **~8–12** (едва слышно) |
| **Toe** | **~18–25** (заметный слеп) |

**Зачем:** акцент на коротких фразах / solo fills; в куплете — пятка. Feedback не на EXP (иначе лента повторов).

### Вариант D — антишип ногой (EQ 4 kHz)

| | |
|--|--|
| **Назначить** | **EQ Guitar EQ1 → 4 kHz** (Gain полосы) |
| **Heel** | **−5…−4** (темнее, безопаснее мониторы) |
| **Toe** | **−3…−1** (как в таблице пресета / чуть ярче) |

**Зачем:** площадка с резкими хэтами — пяткой прибрать [шип](../../terminy/ship.md); для домашнего clean можно не использовать.

Не ведите toe в сильный плюс: яркость через игру и Middle, не через 4k на полу.

### Вариант E — читаемость (EQ 1.6 kHz)

| | |
|--|--|
| **Назначить** | **EQ → 1.6 kHz** |
| **Heel** | **0…+1** (куплет, место вокалу) |
| **Toe** | **+2…+4** (проигрыш / когда тонете) |

**Зачем:** «выйти» в микс на fill без общего Volume bump (меньше конфликтов с вокальным компрессором на пульте).

### Что не ставить на EXP в Folk Clean

| Параметр | Почему |
|----------|--------|
| **AMP Gain** | Легко уйти из clean в edge mid-песней |
| **Bright** | ON/OFF / скачок «стекла» |
| **Gate Threshold** | Хвосты фраз начнут «дырявиться» |
| **Wah Position** | Не характер патча |
| **AMP Volume + VOL вместе** | Два уровня путают; один master на EXP |

### Сочетание с крутилками D/E/F

| Live | Руки (Quick Access) | Нога (EXP) |
|------|---------------------|------------|
| Баланс в бэнде | D = VOL trim **или** | **A** Volume swell (если D занят иначе) |
| Мокро/сухо | E = Room trim | **B** Room Mix |
| Тон | F = 4k / 1.6k | **D** или **E** тот же параметр ногой |

Практично: **EXP = A (Volume swell)** + крутилки D/E/F как в таблице «вход в микс». Либо **EXP = B (Room)** и D остаётся Patch/VOL без swell.

### Мини-чеклист настройки

1. Калибровка EXP (heel / toe упоры).  
2. Выбрать вариант A–E, назначить параметр.  
3. Выставить **min/max** (heel/toe), чтобы toe не клипповал, а heel не давил ниже тишины сцены.  
4. Для A: убедиться, что **VOL до delay/reverb**.  
5. Сыграть куплет–припев в ансамбле; при каше урезать max Mix/Volume на toe.

---

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **Gate Threshold / Ratio** (пауза vs сустейн)  
2. **Dark Twin Gain** (clean vs edge) и **Volume** (уровень)  
3. **Treble / Bright** — если шип; иначе **Middle** / EQ 1.6k  
4. EQ **4k** / **1.6k**  
5. Slapback Mix → Room Mix → CAB Low Cut  

Потом зафиксируйте **D/E/F** как в таблице выше и подгоняйте уже ими.

## Когда пресет не подходит

- Нужен **ритм-чоп** — [Bluegrass Chop](./02-bluegrass-chop.md).  
- Нужен **чистый кельт-chime** — [Celtic Lead](./03-celtic-lead.md).  
- Нужен **яркий лид с лёгким grit** — [Bright Lead](./06-bright-lead.md).  
- Нужен **пад / эмбиент** — [Ambient Pad](./05-ambient-pad.md).  
- Нужна **synth-подложка** — [Synth Bed](./07-synth-bed.md).  
- Нужна **перекличка с OD-гитарой** — [Call & Response](./08-call-response.md).  
- Нужен **регги-ритм (skank)** — [Reggae Skank](./09-reggae-skank.md).
