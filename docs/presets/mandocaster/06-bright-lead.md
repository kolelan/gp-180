# Bright Lead

**Инструмент:** мандокастер (магнит).  
**Стиль:** яркий лид / соло / проигрыш — **не** стерильный clean, но и не рок-стена.  
**Характер:** [edge-of-breakup](../../terminy/edge-of-breakup.md) + блеск **середины**, с жёстким контролем шипа сверху (магнит мандолины и так орёт на ВЧ).

Связанные: [Celtic Lead](./03-celtic-lead.md) (чистый chime), [Rock Crossover](./04-rock-crossover.md) (grit / mid-boost в электро-бэнде), [edge-of-breakup](../../terminy/edge-of-breakup.md), [FRFR](../../terminy/frfr.md).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Лид **с лёгкой грязью**, слышный в бэнде | Чистый кельт-колокольчик → [Celtic Lead](./03-celtic-lead.md) |
| «Яркость» без стеклянного 6–10 kHz | Тёмный grit → [Rock Crossover](./04-rock-crossover.md) |
| Соло / хук поверх аку-ритма и вокала | Чоп → [Bluegrass Chop](./02-bluegrass-chop.md); пад → [Ambient Pad](./05-ambient-pad.md) |

Идея: яркость берём из **1–2.5 kHz** и лёгкого драйва, а «песок» срезаем (**High Cut**, **−6 kHz**, низкий Treble/Presence). Не лезть в Treble «чтобы звонче».

---

## Цепь

```
NR Gate3
→ PRE COMP
→ PRE Micro Boost (опц., CTRL на соло)  — или лёгкий OD 9 Gain низкий
→ N→S Match 35 CL / Foxy CL   (Gain на edge-of-breakup)
   или AMP Foxy 30TB Char Cool / Match-like
→ CAB Foxy 2x12 или Twin 2x12   (OFF, если cab уже в SnapTone)
→ EQ Guitar EQ2
→ MOD OFF (или G-Chorus Depth очень низкий)
→ DLY Digital (Sync 1/8) или короткий Tape
→ RVB Plate (короткий)
→ VOL
```

На [FRFR](../../terminy/frfr.md)/наушниках CAB ON. В INPUT гитарного комбика — часто CAB OFF.

### Зачем каждый блок

| Модуль | Роль здесь |
|--------|------------|
| **Gate3** | Паузы между фразами; не резать сустейн лида ([описание](../../modules/01-nr/gate-3/)) |
| **COMP** | Выровнять атаку медиатора под тремоло-соло |
| **Micro Boost / OD 9** | Чуть «протолкнуть» соло (CTRL), не главный перегруз |
| **Match / Foxy CL** | Открытый тон + минимальный grit на edge |
| **CAB** | Форма; **High Cut жёстче**, чем у Folk — антипесок |
| **EQ2** | Яркость на **1–3 kHz**; вырез **6 kHz** обязателен |
| **Chorus** | Обычно OFF: ширина легко делает «металл» на ВЧ |
| **Digital / Tape** | Воздух лида без длинных хвостов |
| **Plate** | Короткий студийный блеск, не Hall/Shimmer |

Альтернатива «грязнее на грани»: N→S **Match 35 OD** / **Band OD** с **очень низким** Gain (та же логика High Cut / −6k).

## Параметры эффектов — что крутить и зачем

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Лид: не рубить орнаменты. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **20–32** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **220–350** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–100** ≈ ms | стабильность на тремоло |


### PRE — COMP

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Sustain | 25–40 | ровное тремоло |
| Volume | ~55 / unity | |

### PRE — Micro Boost (опц.) или OD 9

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Gain | **15–25** (Micro) / **20–30** (OD 9) | толчок, не fuzz |
| Tone (OD 9) | **40–48** | не открывать верх |
| Volume | около unity | |

На куплет Boost/OD **OFF**, на соло — CTRL ON. Micro Boost: [PRE](../../modules/02-pre/micro-boost/) или [DST](../../modules/04-dst/micro-boost-dst/).

### N→S Match 35 CL / Foxy CL (или AMP)

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Gain** | **25–40** | [edge-of-breakup](../../terminy/edge-of-breakup.md): медиатор чуть «треснет» |
| Bass | 40–50 | + Low Cut в CAB |
| Mid | 50–58 | тело лида |
| **Treble** | **28–38** | магнит уже яркий — не поднимать |
| **Presence** | **30–40** | выше = шип на ВЧ |
| VOL | под патч | |

Если amp «глотает» атаку — чуть Gain вниз, Mid вверх, не Treble.

### CAB Foxy 2x12 / Twin

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Low Cut | ~100–120 Hz | [низ в миксе](../../terminy/niz-v-mikse.md) |
| **High Cut** | **~6.5–8 kHz** | жёстче, чем у Celtic (~8–9): меньше песка |
| Volume | под баланс | |

Не тянуть V30-scoop в плотном аку-бэнде — [почему](../../ir-acoustic/terminy-mic-ir-v30-scoop.md).

### Guitar EQ 2

| Полоса | Значение | Зачем |
|--------|----------|--------|
| 100 Hz | −3…−4 | место басу/бочке |
| **1 kHz** | **+2…+3** | «яркий» лид = читаемая середина |
| 3 kHz | 0…+1 | осторожно: рядом с вокалом |
| **6 kHz** | **−4…−6** | главный антишип мандокастера |

Правило: не хватает звона → **+1 kHz**, не +6 kHz и не Treble.

### MOD
**OFF** база. Если нужен воздух — G-Chorus Depth **≤15–20**, Rate низкий. Глубже — верх «звенит» металлом.

### DLY Digital (или Tape)

| Параметр | Значение |
|----------|----------|
| Mix | **18–25** |
| Time | **1/8**, Sync ON (или короткий Tape) |
| Feedback | низкий (1–2 повтора) |
| Trail | ON удобно на хвостах фраз |

### RVB Plate

| Параметр | Значение |
|----------|----------|
| Mix | **12–18** |
| Decay | короткий–средний |
| High Damp | чуть вверх (тише яркий хвост) |

### VOL
Уровень патча. Свэллы на EXP: VOL **до** Delay/Plate — иначе гасятся хвосты вместе с нотой.

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **Gain** amp (edge, не хайгейн)  
2. **EQ 6 kHz** и/или **CAB High Cut** — если режет ухо  
3. **EQ 1 kHz** — если тонет в бэнде  
4. **Treble / Presence** вниз, не вверх  
5. Delay Mix → Plate Mix  

**CTRL:** Micro Boost / OD 9 ON на соло; опционально Delay Mix↑.

## Когда пресет не подходит

- Нужен **абсолютно чистый chime** без grit — [Celtic Lead](./03-celtic-lead.md).  
- Нужен **рок mid-boost / spring** рядом с электро — [Rock Crossover](./04-rock-crossover.md).  
- Нужен **ритм-чоп** — [Bluegrass Chop](./02-bluegrass-chop.md).  
- Нужен **пад / эмбиент** — [Ambient Pad](./05-ambient-pad.md).  
- Нужна **синтезаторная подложка** в миксе — [Synth Bed](./07-synth-bed.md).  
- Нужна **перекличка с OD-гитарой** — [Call & Response](./08-call-response.md).  
- Ухо всё равно орёт на ВЧ — ещё −6 kHz / High Cut ниже; яркость только через 1 kHz.
