# Synth Bed

**Инструмент:** мандокастер (магнит).  
**Стиль:** фон в миксе — вступление, припев «подкладки», cinematic / indie / ethereal.  
**Характер:** **синтезаторная фактура**: атака медиатора спрятана, слой как pad/drone, мандолина почти не «читается» как щипковый инструмент.

Отличие от [Ambient Pad](./05-ambient-pad.md): там ещё слышна мандолина со свэллами; здесь цель — **неузнаваемый soft-synth bed** под вокал и ритм (тише и уже по спектру).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Фон / подложка **как синтезатор** в миксе | Читаемый лид → [Celtic](./03-celtic-lead.md) / [Bright Lead](./06-bright-lead.md); дуэль с OD → [Call & Response](./08-call-response.md) |
| Долгий зависающий слой под куплет | Гитарный ambient с узнаваемой атакой → [Ambient Pad](./05-ambient-pad.md) |
| Луп Post уже «как pad» | Ритм / чоп → [Bluegrass Chop](./02-bluegrass-chop.md) |

Правило посадки: **тише лида и вокала**, узкая полоса (мало низа, мало шипа 6 kHz) — bed не должен конкурировать с бочкой и сибилянтами.

---

## Цепь

```
NR Gate3
→ PRE OCTA                 (Low Oct умеренно; Dry не в ноль)
→ N→S Dark CL / J-120      (Gain низкий, очень тёмный)
→ CAB Twin 2x12            (High Cut жёстче; OFF на комбике)
→ EQ Guitar EQ2            (вырезать атаку и шип)
→ MOD Auto Swell           (длинный Attack — убить медиатор)
→ MOD Detune               (если один слот MOD — Detune приоритетнее Swell;
                            либо Swell в MOD, Detune заменить чуть сильнее OCTA)
→ Freeze на CTRL           (держать аккорд / ноту)
→ DLY Dual Echo            (длинные, тихие повторы)
→ RVB Shimmer              (Mix низкий в бэнде)
→ VOL                      (уровень bed ниже лид-патчей)
```

На практике в одном **MOD** часто один эффект: выбирайте **Auto Swell** как базу «синтеза»; **Freeze** — если модель позволяет второй слот / вместо Swell на CTRL. **Detune** можно поставить в PRE вместо OCTA, если октава лишняя.

Компактный вариант (меньше DSP):

```
NR → PRE Detune → Dark CL → CAB Twin → EQ2
→ Auto Swell → Dual Echo → Shimmer → VOL
(+ Freeze Activate на CTRL, если слот свободен)
```

## Параметры эффектов — что крутить и зачем

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Synth bed / swell. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **15–28** | порог по паузе без игры |
| **Ratio** | **25–45** | жёсткость глушения |
| **Attack** | **10–40** ≈ ms | открытие под атаку |
| **Release** | **400–1200** ≈ ms | хвост ноты / тишина |
| **Hold** | **80–200** ≈ ms | стабильность на тремоло |


### PRE — OCTA ([карточка](../../modules/02-pre/octa/))

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Low Oct** | **25–40** | «синтезаторный» низ / пятый слой |
| High Oct | 0 или едва | вверх легко орёт |
| **Dry** | **45–60** | не полностью октава-only (каша и лаги) |

Альтернатива «хорал»: PRE **Detune** Wet **30–45**, Detune ±8…15¢ — [Detune](../../modules/09-mod/detune/).  
Альтернатива «квинта-pad»: PRE **Pitch** Hi **+7**, Hi Vol низкий — [Pitch](../../modules/02-pre/pitch/).

### N→S Dark CL (или J-120) + CAB Twin

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Gain | **10–25** | без grit |
| Treble / Presence | **низко** | synth bed ≠ chime |
| Mid | нейтраль / чуть вниз | не есть вокал |
| Low Cut | **~120–150 Hz** | bed не ест бочку/бас |
| **High Cut** | **~6.5–8 kHz** | убрать песок магнита |

### Guitar EQ 2

| Полоса | Значение | Зачем |
|--------|----------|--------|
| 100 Hz | **−4…−6** | место ритм-секции |
| 500 Hz–1 kHz | 0…−2 | не гудеть под вокалом |
| **3–6 kHz** | **−3…−5** | спрятать щипок / шип |
| Volume EQ | ниже лид-патчей | bed = фон |

### MOD — Auto Swell

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Attack** | **длинный** | главное: нет медиаторной атаки = «ключ» синтеза |
| Curve | Exp | мягче старт |

[Auto Swell](../../modules/09-mod/auto-swell/).

### MOD — Freeze (CTRL)

| Параметр | Значение |
|----------|----------|
| Activate | CTRL (зажали аккорд → держим слой) |
| Attack / Release | плавные |
| Volume | под bed, не громче живой игры |

[Freeze](../../modules/09-mod/freeze/).

### DLY Dual Echo

| Параметр | Значение |
|----------|----------|
| Mix A/B | **низкий–средний** (фон, не эхо-соло) |
| Time A ≠ B | размытие, не грид-кельт |
| Feedback | средний; Trail ON |

[Dual Echo](../../modules/10-dly/dual-echo/).

### RVB Shimmer

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Mix** | дома **20–30**; в бэнде **≤10–15** | иначе каша с тарелками |
| Decay | длинный ок для intro | на плотной сцене короче |
| Low/High End | низ хвоста прибрать | |

[Shimmer](../../modules/11-rvb/shimmer/). Замена спокойнее: [Sweet Space](../../modules/11-rvb/sweet-space/).

### VOL

Уровень **ниже** Folk/Celtic/Bright (ориентир −3…−6 дБ ощущением). Bed не должен «выигрывать» микс.

### Looper

Режим **Post** — пишется уже синтезаторный слой. Pre — только если поверх bed будете менять другой патч. См. [лупер](../../manual-ru/05-ekrany.md).

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **Auto Swell Attack** — пока не исчезнет щипок  
2. **EQ 3–6 kHz** / High Cut — если всё ещё «мандолина»  
3. **OCTA Low Oct** или **Detune Wet** — толщина synth  
4. **Shimmer Mix** — место в миксе  
5. **Patch / VOL** — bed тише лида  

**CTRL:** Freeze Activate (держать гармонию); на сцене Shimmer Mix↓.

---

## Когда пресет не подходит

- Нужна **узнаваемая мандолина** (лид, орнаменты, чоп) — любой другой патч пачки.  
- Нужен **ambient, но ещё гитарный** — [Ambient Pad](./05-ambient-pad.md).  
- **Плотный блюграсс / быстрый грид** — хвосты размоют ансамбль.  
- Bed **глотает вокал** — ещё −1…3 kHz, Quiet VOL, Shimmer Mix↓; не поднимайте Treble «для красоты».
