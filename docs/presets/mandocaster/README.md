# Пресеты для мандокастера (мандолина + магнитный датчик)

Мандокастер ближе к «яркой электро-гитаре малого корпуса», чем к пьезо-акустике: много атаки от медиатора, узкий низ, пик в ВЧ. Цель — **не усугублять резкость** и оставить место басу, ритму, вокалу и ударным.

## Общие правила

1. **Компрессия** мягкая (COMP / COMP4) — выровнять чоп, не удушить.
2. **Gate** не агрессивный — иначе обрежется сустейн тремоло.
3. **Gain** низкий; лучше edge-of-breakup, чем хайгейн.
4. **HPF** в CAB Low Cut ~80–120 Гц.
5. **EQ**: резать 3–6 кГц при необходимости; поднимать 1–2 кГц для читаемости.
6. Реверб/дилей короче, чем «гитарная стена» — мандолина и так заполняет верх.

См. также: [EQ в миксе](../../eq-mix/), [рецепты NAM](../../nam-snaptone/recepty/), [акустика + пьезо](../akusticheskaya-mandolina-piezo/).
