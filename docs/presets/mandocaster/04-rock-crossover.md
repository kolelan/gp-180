# Rock Crossover

**Инструмент:** мандокастер (магнит).  
**Стиль:** мандолина в электро-бэнде (бас · ритм-электро · вокал · ударные).  
**Характер:** лёгкий grit, читаемая середина, весенний spring/tape — не metal wall.

См. микс: [eq-mix](../../eq-mix/), [elektro-ansambl](../elektro-ansambl/).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Сесть рядом с электрогитарой | Чистый фолк → Folk Clean |
| Чуть overdrive / mid-boost | Кельт-chime → Celtic Lead |
| Лид/риф в рок-кроссовере | Сухой чоп → Bluegrass Chop |

---

## Цепь

```
NR Gate3 → PRE OD 9 (mid-boost) → N→S Band OD (или AMP L-Star OD)
→ CAB Twin или UK 2x12 → EQ Guitar EQ1 → DLY Tape → RVB Spring → VOL
```

---

## Параметры эффектов — что крутить и зачем

### Gate 3
| Параметр | Значение |
|----------|----------|
| Threshold | 25–40 (мягче chop, жёстче folk-балады при шумном OD) |

### PRE — OD 9 (Tube Screamer-style)
Работает как **mid-boost**, не как главный дист.

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Gain** | **25–35** | толкнуть середину сквозь бэнд |
| Tone | 45–55 | |
| Volume | unity / чуть выше | |

Карточка: [OD 9](../../modules/02-pre/od-9/).

### N→S Band OD / AMP L-Star OD
| Параметр | Значение |
|----------|----------|
| **Gain** | **30–45** — край breakup / лёгкий OD |
| Bass | не задирать (+ Low Cut) |
| Mid / Presence | под читаемость |
| Treble | сдержанно |

### CAB Twin или UK 2x12
| Параметр | Значение |
|----------|----------|
| Low Cut | ~100 Hz |
| High Cut | ~8–9 kHz |
| Для «британского» | UK 2x12; для открытости — Twin |

Без жёсткого V30-scoop в плотном акустическом бэнде — [термины](../../ir-acoustic/terminy-mic-ir-v30-scoop.md).

### Guitar EQ 1
| Полоса | Значение | Зачем |
|--------|----------|--------|
| 125 | **−4** | [низ](../../terminy/niz-v-mikse.md) |
| 400 | 0…−1 | |
| **800** | **+3** | прорезь mid рядом с ритм-гитарой |
| 1.6k | 0…+1 | осторожно: зона мандо-лида |
| 4k | **−2** | |

Ритм-электро должен **отдать** 1.5–2.5 кГц мандолине — [mix-folk](../elektro-ansambl/mix-folk.md).

### DLY Tape
| Параметр | Значение |
|----------|----------|
| Mix | **~15** |
| Feedback | низкий |
| Time | ~1/8 или короткий; Sync по вкусу |
| Trail | ON удобно |

### RVB Spring
| Параметр | Значение |
|----------|----------|
| Mix | **15–20** |
| Decay | короткий–средний |
| Tone | не ярчить |

### VOL
Уровень патча; CTRL см. ниже.

---

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. OD9 Gain → 2. Amp Gain → 3. EQ 800 / 125 → 4. Tape Mix → 5. Spring Mix  

**CTRL:** на соло — OD 9 ON и/или Delay Mix↑.

---

## Когда пресет не подходит

- Нужен **блюграсс-чоп** без grit — [Bluegrass Chop](./02-bluegrass-chop.md).  
- Нужен **тёмный americana без перегруза** — [Folk Clean](./01-folk-clean.md) или пьезо-патчи.  
- Нужен **чистый кельт-лид** — [Celtic Lead](./03-celtic-lead.md).  
- Нужен **яркий лид на грани** без рок-spring — [Bright Lead](./06-bright-lead.md).
