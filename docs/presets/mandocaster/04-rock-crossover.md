# Rock Crossover

**Роль:** мандолина в электро-бэнде с басом, ритм-гитарой, вокалом, барабанами.

```
NR Gate3 → PRE OD9 (как mid-boost) → N→S Band OD или AMP L-Star OD
→ CAB Twin или UK 2x12 → EQ Guitar EQ1 → DLY Tape → RVB Spring → VOL
```

| Параметр | Значение |
|----------|----------|
| OD9 Gain | 25–35 (буст середин) |
| Amp/ST Gain | 30–45 |
| EQ 125 | −4 |
| EQ 800 | +3 |
| EQ 4k | −2 |
| Tape Mix | 15 |
| Spring Mix | 15–20 |

См. раздел [eq-mix](../../eq-mix/) — ритм-гитара должна отдавать 1.5–2.5 кГц.
