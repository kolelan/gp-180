# Call & Response

**Инструмент:** мандокастер (магнит).  
**Стиль:** соло **вопрос–ответ** с перегруженной электрогитарой (фраза гитары → фраза мандо / наоборот).  
**Характер:** плотная читаемая середина, лёгкий grit под энергию OD-гитары, **короткие** FX — паузы между репликами должны быть слышны.

См. посадку: [mix-folk](../elektro-ansambl/mix-folk.md), [Rock Crossover](./04-rock-crossover.md) (общий рок в бэнде), [Bright Lead](./06-bright-lead.md) (яркий лид без дуэли).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Перекличка соло с **overdrive / distortion** гитарой | Чистый кельт → [Celtic Lead](./03-celtic-lead.md) |
| Фразы чётко входят/выходят, без «хвоста на чужой вопрос» | Длинный ambient / synth → [Ambient](./05-ambient-pad.md) / [Synth Bed](./07-synth-bed.md) |
| Мандолина **прорезает** рядом с толстым OD | Спокойный sit-in в куплете → [Rock Crossover](./04-rock-crossover.md) |

Идея: энергия близка к электро, но спектр **не копирует** гитару — мандо на **~1.6–2.5 кГц**, гитаре лучше отдать низ/середину и не дублировать 2 кГц на полном Gain.

---

## Цепь

```
NR Gate3
→ PRE COMP4              (ровные «ответы», медиатор не прыгает)
→ PRE OD 9               (mid-boost; Gain умеренный)
→ N→S Band OD / ToneK OD / Match 35 OD   (Gain edge…лёгкий OD)
   или AMP L-Star OD
→ CAB UK 2x12 или Twin   (UK — плотнее «против» Marshall-like гитары)
→ EQ Guitar EQ1
→ MOD OFF
→ DLY Digital или Slapback   (очень короткий / 1–2 повтора)
→ RVB Room                   (Mix минимум; не Spring-стена)
→ VOL
```

На [FRFR](../../terminy/frfr.md)/наушниках CAB ON. В комбик — часто CAB OFF.

### Зачем каждый блок

| Модуль | Роль здесь |
|--------|------------|
| **Gate3** | Чуть плотнее folk: тишина между репликами, без «шипа в паузе» |
| **COMP4** | Одинаковая громкость коротких ответов |
| **OD 9** | Протолкнуть mid сквозь OD-гитару |
| **Band OD** | Согласовать энергию с перегрузом партнёра, не metal wall |
| **EQ1** | Пик читаемости + антишип; низ отдать гитаре/басу |
| **Короткий delay** | Чуть воздуха; длинный echo съест «вопрос» гитары |
| **Room** | Минимальная сцена, не хвост на чужую фразу |

## Параметры эффектов — что крутить и зачем

### Gate 3

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Threshold | **28–40** | Плотнее Ambient/Celtic; паузы чище |
| Release | средний | не рубить конец фразы |

### PRE — COMP4

| Параметр | Значение |
|----------|----------|
| Sustain / Ratio | умеренно (фразы ровные, не «квадрат») |
| Attack | не слишком медленный — атака ответа должна щёлкнуть |

### PRE — OD 9

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Gain** | **30–40** | mid-boost сквозь гитару |
| Tone | **42–50** | не открывать песок (гитара уже яркая) |
| Volume | unity / +чуть | |

[OD 9](../../modules/02-pre/od-9/).

### N→S Band OD / ToneK OD (или AMP L-Star OD)

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Gain** | **35–50** | [edge-of-breakup](../../terminy/edge-of-breakup.md)…лёгкий OD; не догонять хайгейн гитары уровнем грязи |
| Bass | сдержанно | толстый низ = каша с OD |
| Mid | вверх | суть переклички |
| Treble / Presence | сдержанно | шип vs гитарный crunch |
| + CAB Low Cut | **~100–120 Hz** | |

Если гитара очень «шершавая» — мандо **чуть чище** (Gain ниже): контраст тембров читается лучше, чем два одинаковых диста.

### CAB UK 2x12 / Twin

| Параметр | Значение |
|----------|----------|
| High Cut | **~7.5–9 kHz** |
| Low Cut | ~100–120 Hz |

### Guitar EQ 1

| Полоса | Значение | Зачем |
|--------|----------|--------|
| 125 | **−4…−5** | низ гитаре/басу |
| **800** | **0…+2** | осторожно: зона толстого OD |
| **1.6k** | **+3…+4** | главный «ответ» в миксе |
| 4k | **−1…−2** | |
| **6k / верх** | **−3…−4** | антишип рядом с ярким OD |

Договорённость с гитаристом: на время фраз мандо — чуть **убрать 1.5–2.5 кГц** / Gain на его пресете (или Volume вниз). Иначе оба орут в одном кармане.

### DLY Digital / Slapback

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Mix | **10–18** | |
| Time | **короткий** или 1/8 с **низким** Feedback | 1 повтор max в плотной перекличке |
| Feedback | **очень низкий** | второй «хвост» = чужой вопрос |

[Slapback](../../terminy/slapback.md) уместен; длинный Tape/Dual — нет.

### RVB Room

| Параметр | Значение |
|----------|----------|
| Mix | **8–14** |
| Decay | короткий |

Spring/Hall/Shimmer здесь мешают диалогу фраз.

### VOL

Уровень **сопоставим** с соло гитары в мониторе (не тише «ответа»). CTRL — небольшой bump Volume / OD на свою реплику.

## Игра (вопрос–ответ)

1. Гитара фраза → **пауза** (гейт/тишина) → мандо фраза той же длины/энергетики.  
2. Не играйте внакладку на длинном delay гитариста.  
3. Короткие «выдохи» медиатором; тремоло-стена легче путается с сустейном OD.  
4. На CTRL: OD 9 / Micro Boost только на **свою** реплику, в паузе — OFF.

---

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **EQ 1.6 kHz** — слышно ли «ответ» рядом с OD  
2. **OD 9 / Amp Gain** — энергия без каши  
3. **EQ 125 / Low Cut** — гул с гитарой  
4. **6 kHz** — резкость  
5. **Delay/Room Mix** вниз, если фразы слипаются  

**CTRL:** Volume↑ или OD 9 ON на ответ; Delay не поднимать.

---

## Когда пресет не подходит

- Нужен **общий рок sit-in** (куплет/рифф), не дуэль соло — [Rock Crossover](./04-rock-crossover.md).  
- Нужен **яркий лид без дуэли с OD** — [Bright Lead](./06-bright-lead.md).  
- Нужен **чистый chime** — [Celtic Lead](./03-celtic-lead.md).  
- Гитара на **хайгейне** заливает всё — договоритесь о частотах/громкости; не лечите только +Treble на мандо.
