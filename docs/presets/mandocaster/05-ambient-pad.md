# Ambient Pad

**Роль:** текстуры, вступления, лупы.

```
NR → PRE Saturate (Mix низкий) → N→S Dark CL → CAB Twin
→ EQ → MOD Auto Swell + Freeze (по CTRL) → DLY Dual Echo → RVB Shimmer → VOL
```

| Параметр | Значение |
|----------|----------|
| Saturate Mix | 20 |
| Auto Swell Attack | средний |
| Shimmer Mix | 25 (на сцене с бэндом ↓ до 12) |
| Dual Echo | разные Time A/B, Mix умеренный |

Looper: **Post** для записи готового пада; играйте поверх другим патчем (Pre-луп).
