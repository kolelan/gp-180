# Reggae Skank

**Инструмент:** мандокастер (магнит).  
**Стиль:** регги / ska-ритм — **оффбит-чоп (skank)** как «реггийная окраска» грида.  
**Роль в миксе:** электрогитара играет **фоном** (пад, dub-линия, тихий bed); мандокастер даёт **щелчок offbeat** и реггийный характер ритма.

См.: [mix-reggae](../elektro-ansambl/mix-reggae.md), электро-регги [clean](../elektro-ansambl/03-reggae-clean.md) / [edge](../elektro-ansambl/04-reggae-edge.md), блюграсс-чоп [Bluegrass Chop](./02-bluegrass-chop.md) (другой жанр, похожая сухость).

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Ритм-оффбит, «chick» поверх тихого гитарного фона | Мелодии / fill → [Celtic](./03-celtic-lead.md) / [Folk](./01-folk-clean.md) |
| Реггийная окраска грида именно мандолиной | Блюграсс backbeat → [Bluegrass Chop](./02-bluegrass-chop.md) |
| Чёткие паузы one-drop / rockers | Длинные хвосты → Ambient / Synth |

Идея: **коротко, сухо, в паузах тишина** — как классический skank, но выше по тембру. Бас и кик остаются главными по низу; гитара не должна дублировать тот же offbeat громко.

---

## Цепь

```
NR Gate2
→ PRE COMP4
→ AMP J-120 CL / Dark Twin   (или N→S J-120 Bright / Dark CL)
→ CAB J-120 2x12 / Twin
→ EQ Guitar EQ2
→ MOD OFF                    (опц. лёгкий Sine Trem Depth низкий — rarely)
→ DLY OFF
→ RVB Room (Mix минимум)
→ VOL
```

Чистый solid-state / тёмный Twin = ровный chick без ламповой каши. Gain без [edge](../../terminy/edge-of-breakup.md), если только не dub-вариант «чуть грязи».

---

## Параметры эффектов — что крутить и зачем

### Gate 2

| Параметр | Значение | Зачем |
|----------|----------|--------|
| **Threshold** | **40–55** | паузы между offbeat — суть регги |
| Attack | короткий | |
| Release | средний | не «сюр» хвост в тишину |

Плотнее Folk; близко к электро [Reggae Clean](../elektro-ansambl/03-reggae-clean.md).

### PRE — COMP4

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Sustain | **~35–45** | ровный chick |
| **Attack** | **быстрый** | словить атаку медиатора |
| Volume | под уровень | |

### AMP / N→S + CAB

| Параметр | Значение | Зачем |
|----------|----------|--------|
| Gain | **15–30** | clean skank |
| Bright / Treble | сдержанно | мандо и так щелкает |
| Bass | низко | место **басу** (в регги бас главный) |
| Low Cut | **~100–120 Hz** | |
| High Cut | **~7.5–8.5 kHz** | не есть rim/хэт |

Без длинной сатурации — иначе offbeat «смазывается».

### Guitar EQ 2

| Полоса | Значение | Зачем |
|--------|----------|--------|
| 100 Hz | **−4…−5** | бас / кик |
| 500 Hz | −1…0 | |
| **1 kHz** | **+2…+3** | читаемый chick |
| 3 kHz | 0…−1 | не бить по snare/rim |
| **6 kHz** | **−3…−4** | антишип; rim остаётся ударным |

Мандолина здесь **ярче серединой**, не верхом «[блюграсс-шипа](../../terminy/ship.md)».

### MOD / DLY
**OFF** — delay рвёт грид и маскирует паузы.

Опция dub-еффекта только на **fill** отдельным патчем/CTRL (короткий delay), не на постоянный skank.

### RVB Room

| Параметр | Значение |
|----------|----------|
| Mix | **5–12** |
| Decay | короткий |

### VOL
Уровень **выше** фоновой гитары: skank должен читаться как ритм-окраска. Гитара bed — тише и без того же offbeat.

## Игра и посадка

1. Чоп на **offbeat** (&) / skank — короткие приглушённые аккорды или двойки; не тремоло-стена.  
2. **Гитара фоном:** длинные ноты, тихий pad, dub-эхо — **не** громкий параллельный skank в тех же долях.  
3. Бас ведёт мелодию низа; не забивайте 80–120 Гц.  
4. One-drop: паузы важнее «количества нот».

CTRL: на короткий fill — Volume↑ и чуть отпустить Gate; сразу вернуть порог для куплета.

---

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. **Gate Threshold** — слышны ли паузы  
2. **COMP4 Attack** — щёлкает ли chick  
3. **EQ 1 kHz / 6 kHz** — читаемость vs шип  
4. **Low Cut / 100 Hz** — место басу  
5. **Room Mix** ≈ минимум  

**CTRL:** Volume / ослабить Gate на fill.

---

## Когда пресет не подходит

- Нужен **блюграсс-чоп** / фолк backbeat — [Bluegrass Chop](./02-bluegrass-chop.md).  
- Нужны **мелодии / лид** поверх регги — [Celtic](./03-celtic-lead.md) / [Bright Lead](./06-bright-lead.md); skank выключите.  
- Гитара тоже громко рубит offbeat — получится двойной skank; уберите её чоп или этот патч.  
- Нужен **мокрённый dub-хвост** на мандо — не база; отдельный лид/fill, не ритм-патч.
