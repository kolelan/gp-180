# Bluegrass Chop

**Роль:** ритм-чоп, backbeat, плотность без каши.

```
NR Gate2 → PRE COMP4 (Attack быстрее) → AMP J-120 CL или N→S J-120 Bright
→ CAB J-120 2x12 → EQ Guitar EQ2 → MOD off → DLY off → RVB Room Mix очень низкий → VOL
```

| Параметр | Значение |
|----------|----------|
| Gate Threshold | 40–50 (чёткие паузы) |
| COMP4 Sustain / Attack | 35 / быстрый |
| Gain | 15–30 |
| EQ 100 Hz | −4 |
| EQ 500 Hz | −1 |
| EQ 1 kHz | +2 |
| EQ 3 kHz | 0…−2 |
| EQ 6 kHz | −4 |
| Room Mix | 5–12 |

**Идея:** максимум атаки и минимума хвостов — ударные и аку-гитара дышат.
