# Bluegrass Chop

**Инструмент:** мандокастер (магнит).  
**Стиль:** блюграсс / фолк-ритм, backbeat, «щёлкающий» чоп.  
**Характер:** сухой, агрессивный по атаке, почти без хвостов.

---

## Зачем этот патч

| Нужно | Не этот патч |
|-------|----------------|
| Ритм, паузы между ударами | Мелодии с delay → Celtic Lead / Folk Clean |
| Чёткий грид с аку-гитарой и ударными | Ambient / Plate |

Идея: **максимум атаки, минимум пространства**.

---

## Цепь

```
NR Gate2 → PRE COMP4 → AMP J-120 CL (или N→S J-120 Bright)
→ CAB J-120 2x12 → EQ Guitar EQ2
→ MOD off → DLY off → RVB Room (Mix очень низкий) → VOL
```

JC / solid-state clean = ровный, без ламповой «каши».

---

## Параметры эффектов — что крутить и зачем

### Gate 2
| Параметр | Для Chop |
|----------|----------|
| **Threshold** | **40–50** — чёткие паузы между ударами |
| Attack | короткий |
| Release | средний |

Жёстче, чем у Folk/Celtic: пауза важнее сустейна.

### COMP4
| Параметр | Для Chop |
|----------|----------|
| **Sustain** | **~35** |
| **Attack** | **быстрый** — словить пик медиатора |
| Volume | под уровень |
| Clipping | не задирать |

### AMP J-120 CL / N→S J-120 Bright
| Параметр | Для Chop |
|----------|----------|
| **Gain** | **15–30** |
| Bright | осторожно; часто OFF или слабый |
| EQ amp | средне; яркость лучше снимать EQ2 6k |

### CAB J-120 2x12
| Параметр | Для Chop |
|----------|----------|
| Low Cut | ~100–120 Hz |
| High Cut | **~8 kHz** (жёстче, чем у Folk) |

### Guitar EQ 2
| Полоса | Значение | Зачем |
|--------|----------|--------|
| 100 Hz | **−4** | [низ](../../terminy/niz-v-mikse.md) бочке/басу |
| 500 Hz | **−1** | |
| **1 kHz** | **+2** | щелчок чопа |
| 3 kHz | 0…−2 | не есть вокал |
| **6 kHz** | **−4** | антишип |

### MOD / DLY
**OFF** — иначе ритм плывёт.

### RVB Room
| Параметр | Значение |
|----------|----------|
| Mix | **5–12** только |
| Decay | короткий |

---

## Сначала крутите это

Если патч не сел сразу — меняйте **только эти** параметры, в указанном порядке. Остальное можно не трогать.

1. Gate Threshold → 2. COMP4 Attack/Sustain → 3. EQ 1k / 6k → 4. Room Mix ≈ минимум  

**CTRL:** на fill чуть поднять Volume и ослабить Gate (Threshold ниже / Release длиннее), чтобы не срезать концы фраз.

---

## Когда пресет не подходит

- Нужен **лид с delay / plate** — [Celtic Lead](./03-celtic-lead.md).  
- Нужна **баллада / тёплый чистый** — [Folk Clean](./01-folk-clean.md).  
- Нужен **пад / эмбиент** — [Ambient Pad](./05-ambient-pad.md).  
- Нужен **регги offbeat / skank** — [Reggae Skank](./09-reggae-skank.md).
