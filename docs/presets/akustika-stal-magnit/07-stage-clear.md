# Stage Clear

**Роль:** громкие мониторы / камерная сцена — меньше гула и «коробки» магнита.

```
NR Gate3 → PRE COMP4 → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → RVB Room Mix минимальный → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Сцена / anti-FB: суше паузы. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **35–50** | порог по паузе без игры |
| **Ratio** | **50–70** | жёсткость глушения |
| **Attack** | **5–15** ≈ ms | открытие под атаку |
| **Release** | **100–200** ≈ ms | хвост ноты / тишина |
| **Hold** | **20–50** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| COMP4 | Sustain | 30–40 |
| AC Pre | Tone | 40–48 |
| AC Pre | EQ Freq | ~250–400 Hz (под гул) |
| AC Pre | EQ Q | уже |
| AC Pre | EQ Gain | 42–47 |
| CAB | Low / High Cut | 120–140 Hz / 7.5–8.5 kHz |
| EQ2 | 100 / 500 / 6k | −5 / −3 / −3 |
| Room | Mix | 5–12 |
| Patch Volume | | чуть ниже Natural |

Магнит howlit реже пьезо; обычно мешает низ-середина — точечный вырез важнее глубокого High Cut.
