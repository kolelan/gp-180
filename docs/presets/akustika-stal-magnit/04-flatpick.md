# Flatpick Drive

**Роль:** медиаторный фолк/кантри: атака, щёлк, лёгкая плотность без электро-кранча.

```
NR Gate3 → PRE COMP4 → AMP AC Pre → CAB Bird
→ EQ Guitar EQ1 → DLY Slapback → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP4 | Sustain / Attack | 40 / быстрый |
| AC Pre | Tone | 50–58 |
| AC Pre | EQ Freq / Gain | ~400 Hz / 47–49 |
| CAB Bird | Low / High Cut | 100 Hz / 9 kHz |
| EQ1 | 800 / 1.6k / 4k | +2 / +2 / −1 |
| Slapback | Mix / Time | 12–18 / 90–120 ms |
| Room | Mix | 10–18 |

Нужен чуть grit: PRE **Saturate** Mix 15–25 вместо OD-педали.
