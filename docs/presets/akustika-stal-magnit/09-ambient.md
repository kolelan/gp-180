# Ambient Pad

**Роль:** подложки и лупы на магнитном акустическом канале.

```
NR → PRE COMP → AMP AC Pre → CAB OM → EQ Guitar EQ2
→ MOD G-Chorus / Auto Swell → DLY Vintage Rack → RVB Sweet Space → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Chorus | Depth | 20–35 |
| Auto Swell | Attack | средний (для падов) |
| Vintage Rack | Mix / Mod | 22–32 / немного |
| Sweet Space | Mix | 22–35 |
| EQ2 | 6k | −3 |
| CAB High Cut | | 8–9 kHz |

Looper: **Pre** — сухой bed; **Post** — готовый космос.
