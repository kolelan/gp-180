# Fingerstyle Dynamic

**Роль:** перебор, динамика пальцев, читаемый бас без гула.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ1 → DLY Slapback тихо → RVB Plate → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 20–35 |
| AC Pre | Tone | 50–58 |
| AC Pre | EQ Freq / Gain | ~220 Hz / 47–50 (или нейтраль) |
| CAB | Low / High Cut | 80–100 Hz / 10 kHz |
| EQ1 | 125 / 400 / 1.6k / 4k | −1 / −1 / +2 / 0 |
| Slapback | Mix / Time | 8–12 / 80–110 ms |
| Plate | Mix | 12–20 |

**CTRL:** +Volume или лёгкий +1.6 kHz на мелодический проход.
