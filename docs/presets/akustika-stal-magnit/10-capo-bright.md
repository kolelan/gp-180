# Capo Bright

**Роль:** каподастр / игра выше 5-го лада: магнит становится тоньше и ярче — сгладить шип, сохранить щёлк струн.

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ2 → DLY Digital тихо → RVB Plate → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 25–40 |
| AC Pre | Tone | 42–50 (не ярче Natural) |
| AC Pre | EQ Freq / Gain | ~2–3.5 kHz / 46–49 |
| CAB | Low / High Cut | 100 Hz / 8 kHz |
| EQ2 | 1k / 3k / 6k | +1 / −2 / −4 |
| Delay | Mix | 10–15 |
| Plate | Mix | 12–20 |

Если без капы тон кажется тёмным — вернитесь к [Natural](./01-natural.md); этот патч специально «придерживает» верх.
