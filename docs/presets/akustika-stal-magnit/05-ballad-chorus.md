# Ballad Chorus

**Роль:** медленные песни, ширина без «пьезо-шиммера».

```
NR Gate3 → PRE COMP → AMP AC Pre → CAB OM
→ EQ Guitar EQ1 → MOD G-Chorus → DLY Digital тихо → RVB Hall → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 30–45 |
| AC Pre | Tone | 48–55 |
| CAB | Cuts | 95 Hz / 9.5 kHz |
| EQ1 | 400 / 4k | +1 / −2 |
| Chorus | Depth / Rate | 20–30 / средний |
| Delay | Mix | 10–15 |
| Hall | Mix | 20–30 |

Chorus на CTRL: в куплет суше, в припев шире.
