# Soft Jazz

**Роль:** тёплые аккорды, балладный джаз/свинг на акустике с магнитом.

```
NR Gate3 → PRE Saturate → AMP J-120 CL (или AC Pre)
→ CAB J-120 2x12 или OM → EQ Guitar EQ1
→ MOD Detune еле → DLY Sweet Echo → RVB Room → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Мягкий сустейн / clean. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **22–30** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **200–320** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–90** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix | 15–30 |
| J-120 / AC Pre | Gain/Tone | низкий / мягкий |
| Bright (если J-120) | | OFF |
| CAB | High Cut | 7.5–8.5 kHz |
| EQ1 | 400 / 4k | +1 / −4 |
| Detune | Wet / cents | 12–20 / ±5…8 |
| Sweet Echo | Mix | 10–15 |
| Room | Mix | 18–28 |
