# Soft Jazz

**Роль:** тёплые аккорды, балладный джаз/свинг на акустике с магнитом.

```
NR Gate3 → PRE Saturate → AMP J-120 CL (или AC Pre)
→ CAB J-120 2x12 или OM → EQ Guitar EQ1
→ MOD Detune еле → DLY Sweet Echo → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix | 15–30 |
| J-120 / AC Pre | Gain/Tone | низкий / мягкий |
| Bright (если J-120) | | OFF |
| CAB | High Cut | 7.5–8.5 kHz |
| EQ1 | 400 / 4k | +1 / −4 |
| Detune | Wet / cents | 12–20 / ±5…8 |
| Sweet Echo | Mix | 10–15 |
| Room | Mix | 18–28 |
