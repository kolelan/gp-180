# Folk — Clean

**Роль электро:** прозрачный аккомпанемент / арпеджио рядом с аку-ритмом и мандолиной.

```
NR Gate3 → PRE COMP → AMP Dark Twin (или N→S Dark CL)
→ CAB Twin 2x12 → EQ Guitar EQ1 → DLY Slapback тихо → RVB Room → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Мягкий сустейн / clean. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **22–30** | порог по паузе без игры |
| **Ratio** | **40–50** | жёсткость глушения |
| **Attack** | **8–15** ≈ ms | открытие под атаку |
| **Release** | **200–320** ≈ ms | хвост ноты / тишина |
| **Hold** | **50–90** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| COMP | Sustain | 25–35 |
| AMP / ST | Gain | 20–35 |
| Bright | | OFF |
| Bass / Mid / Treble | | 45 / 52 / 42 |
| CAB | Low / High Cut | 90 Hz / 9 kHz |
| EQ1 | 125 / 400 / 800 / 1.6k / 4k | −3 / −1 / +1 / +1 / −2 |
| Slapback | Mix | 8–15 |
| Room | Mix | 12–20 |

Не конкурируйте с мандолиной на 2–3 кГц — при конфликте EQ1 1.6k вниз.
