# Folk — Edge (лёгкий OD)

**Роль:** лёгкий grit на припев / лид без «рок-стены».

```
NR Gate3 → PRE OD 9 (Gain низкий) → AMP Tweedy или Foxy 30N
→ CAB Twin или Foxy 2x12 → EQ Guitar EQ1 → DLY Tape тихо → RVB Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| OD 9 | Gain / Tone | 25–40 / 45–55 |
| AMP | Gain | 25–40 |
| EQ1 | 800 / 4k | +2 / −2 |
| Tape | Mix | 10–18 |
| Room | Mix | 12–18 |

На CTRL можно вешать OD 9: куплет clean (модуль OFF), припев — лёгкий push.
