# Микс: регги-ансамбль

| Партия | Приоритет | Заметки |
|--------|-----------|---------|
| Бас | Главный мелодический низ, часто выше по уровню, чем в фолке | Не резать 80–120 Гц у баса «гитарным» HPF на его канале |
| Аку-ритм | Опционально / перкуссивно или копия skank | Если оба режут offbeat — электро суше, аку тише |
| Электро | Offbeat: коротко, без реверб-хвоста | Gate помогает паузам |
| Мандолина | Мелодические fill / unison с вокалом | Не играть постоянный chop поверх электро-skank |
| Ударные | Rim / snare на 2–4, хэт | Электро High Cut; не маскировать rim щелчком 4–6 кГц |

One-drop / rockers: электро Volume стабильный; соло — отдельным CTRL bump.
