# Reggae — Clean

**Роль электро:** чистый skank / offbeat chord chops (часто с Mute).

```
NR Gate2 → PRE COMP4 → AMP J-120 CL (или Dark Twin)
→ CAB J-120 2x12 → EQ Guitar EQ2 → MOD off → DLY off → RVB Room очень низкий → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Gate2 | Threshold | 40–55 (чёткие паузы) |
| COMP4 | Sustain / Attack | 40 / быстрый |
| AMP | Gain / Bright | 15–30 / OFF или лёгкий |
| CAB | Low / High Cut | 100 Hz / 8 kHz |
| EQ2 | 100 / 500 / 1k / 3k / 6k | −4 / −1 / +2 / −1 / −3 |
| Room | Mix | 5–12 |

Середина читаемая (1 кГц), низ отдан басу и кику. Без длинного delay — рвёт регги-грид.
