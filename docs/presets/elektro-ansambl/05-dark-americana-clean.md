# Dark Americana — Clean

**Роль:** тёмный, открытый clean: баллады, дроны, «пыль и пространство». Рядом с акустикой и мандолиной — без блеска pop-country.

```
NR Gate3 → PRE Saturate (Mix низкий) → AMP Bellman 59N / Dark Twin
→ CAB Bellman 2x12 или Twin → EQ Guitar EQ1
→ DLY Tape тихо → RVB Spring / Room → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| Saturate | Mix / High Cut | 15–30 / вниз |
| AMP | Gain | 20–35 |
| Bass / Mid / Treble | | 50 / 48 / 35–42 |
| Presence (если есть) | | 30–40 |
| CAB | Low / High Cut | 85 Hz / 7.5–8.5 kHz |
| EQ1 | 400 / 1.6k / 4k | +1 / 0 / −4 |
| Tape | Mix | 12–20 |
| Spring / Room | Mix | 18–28 |

Темнее, чем folk-clean: меньше воздуха на 4–6 кГц.
