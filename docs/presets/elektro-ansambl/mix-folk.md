# Микс: фолк-ансамбль

Состав: бас · аку-ритм · электро · мандолина · ударные.

| Партия | Частотный приоритет | Громкость / FX |
|--------|---------------------|----------------|
| Бас | 60–200 Гц, панч ~600–800 | Сухо; без длинного реверба |
| Акустика ритм | 200–500 Гц тело, атака ~3–5 кГц мягко | Пьезо/магнит-пресеты из akustika-*; Room низкий |
| Электро clean/edge | 800 Гц–2 кГц | Без толстого низа; delay короткий |
| Мандолина | 1.5–3 кГц, мало низа | Не дублировать электро на 2 кГц |
| Ударные | Кик/снейр + хэт | Электро и мандо: High Cut, без длинного Shimmer |

**Сцена:** электро не громче аку-ритма в куплете; мандолина может быть выше на мелодии.
