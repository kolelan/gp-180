## Общие правила электро в этом бэнде

1. Gain низкий — не перекрикивать акустику и мандолину.
2. [HPF](../../terminy/hpf.md) ~80–100 Гц; середину делить: аку-ритму 200–500 Гц, электро ~800 Гц–2 кГц, мандолине 1.5–3 кГц.
3. Delay/reverb короче, чем в «гитарном» соло-патче.
4. CTRL: Solo bump (Micro Boost / +Mix Delay) — не постоянный хайгейн.
