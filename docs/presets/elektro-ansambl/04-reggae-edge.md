# Reggae — Edge (лёгкий OD)

**Роль:** чуть толще skank / лёгкий lead-fill; всё ещё «не рок».

```
NR Gate2 → PRE COMP → DST Blues OD или Penesas (Gain низкий)
→ AMP Match CL / SUPDual CL → CAB Twin 2x12
→ EQ Guitar EQ1 → RVB Room низкий → VOL
```

| Блок | Параметр | Значение |
|------|----------|----------|
| OD | Gain | 20–35 |
| AMP | Gain | 20–35 |
| EQ1 | 125 / 800 / 4k | −4 / +2 / −2 |
| Room | Mix | 8–15 |

CTRL: OD только на bridge/solo fill. Основной куплет — [Clean](./03-reggae-clean.md).
