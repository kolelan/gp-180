# Dark Americana — Edge (лёгкий OD)

**Роль:** слегка надломленный преамп — припев, lead, текстура поверх акустики; не marshall-wall.

```
NR Gate3 → PRE COMP → DST Penesas или Force (LP, Gain низкий)
→ AMP UK 45 / Tweedy / Flagman 1 (низкий Gain)
→ CAB UK 2x12 или Twin → EQ Guitar EQ1
→ DLY Tape → RVB Plate / Hall Mix умеренный → VOL
```

### Gate 3 ([описание](../../modules/01-nr/gate-3/))

Edge/OD: паузы чуть суше. Единицы и логика ручек — в карточке.

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| **Threshold** | **25–40** | порог по паузе без игры |
| **Ratio** | **45–60** | жёсткость глушения |
| **Attack** | **5–15** ≈ ms | открытие под атаку |
| **Release** | **120–250** ≈ ms | хвост ноты / тишина |
| **Hold** | **30–70** ≈ ms | стабильность на тремоло |


| Блок | Параметр | Значение |
|------|----------|----------|
| OD | Gain | 25–40 |
| AMP | Gain | 30–45 |
| Treble / Presence | | сдержанно |
| EQ1 | 125 / 800 / 4k | −3 / +2 / −3 |
| Tape | Mix / Feedback | 15–25 / низкий |
| Plate | Mix | 15–25 |

CTRL: OD + чуть Delay Mix на соло. Куплет — [Clean](./05-dark-americana-clean.md).
