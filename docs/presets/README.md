# Пресеты проекта

## Примечание об электрогитаре

Большинство **заводских (factory) патчей GP-180** изначально заточены под **электрогитару** (clean / crunch / hi-gain, amp+cab, SnapTone). В этом проекте мы **намеренно не дублируем** полный каталог электрогитарных «всего и сразу»: предполагается, что пользователь может самостоятельно перебрать заводские звуки (New GEN., Dark Clean, Foxy OD, UK800 и т.д.) и SnapTone.

Здесь акцент на:

- **мандолину** (магнит / пьезо) и **акустическую гитару** (магнит / пьезо);
- посадку в микс (бас, ритм, вокал, ударные);
- **узкий набор электрогитары** — только **чистые** и **слегка перегруженные** тона под конкретный **ансамбль**.

Именование наборов: всегда **инструмент · (опции) · датчик**, без общей подписи «Акустика + пьезо».
