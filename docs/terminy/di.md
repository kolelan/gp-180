# DI (Direct Input / Direct Injection)

**DI** — подача **инструментального сигнала напрямую** в микшер / интерфейс / [PA](./pa.md), **минуя** микрофон перед комбиком.

Часто говорят «отдать DI» = «дать чистую линию с процессора/преампа звукорежиссёру».

## Откуда название

**Direct Injection / Direct Input** — «впрыск» сигнала сразу в линию. Классический **DI-box** согласует высокоомный инструмент с микрофонным входом пульта (XLR, баланс).

## DI с GP-180 на практике

С GP-180 «DI» обычно значит:

```
гитара → GP-180 (AMP+CAB или акустическая цепь) → OUT L/R (линейный)
       → вход пульта / интерфейса звукорежиссёра
```

То есть в зал уходит **уже обработанный** моделлером сигнал, а не микрофон с кабинета на сцене.

Иногда DI = ещё «суше» (меньше кабинета/капель), но в разговоре с FOH по GP-180 чаще имеют в виду **линейный выход процессора**, а не обязательно «полностью dry».

## DI vs микрофон с комбика

| | DI с GP-180 | Микрофон у комбика |
|--|-------------|-------------------|
| Сигнал | линия с процессора | акустика комнаты + динамик |
| Окраска | то, что в патче (CAB/IR) | комбик + микрофон + сцена |
| Удобство | стабильно, мало «утечек» | живой воздух, но больше feedback |

На [FRFR](./frfr.md) логика близка к DI: полный сигнал из коробки процессора.

## Не путать

- **DI** — способ **маршрута** сигнала (прямо в пульт).  
- **Dry** — без эффектов. DI может быть и с reverb в патче.  
- **XLR DI-box** — железо между гитарой и пультом; у GP-180 роль «готовности к пульту» часто выполняет сам линейный OUT (иногда всё же нужен DI-box под конкретный вход FOH).
