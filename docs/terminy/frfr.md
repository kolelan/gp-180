# FRFR

**FRFR** = **Full Range, Flat Response** — полнодиапазонная акустика с **относительно ровной** АЧХ.

## Зачем термин

Классический **гитарный комбик** сам сильно красит звук (динамик + корпус = уже «кабинет»).  
**FRFR** (студийные мониторы, многие активные «акустические» колонки, часть stage wedges / FRFR-боксов для моделлеров) рассчитан на то, чтобы **воспроизвести то, что уже сделал процессор**, без сильной дополнительной окраски.

Поэтому на GP-180 в пресетах пишут «ориентир под **FRFR**/наушники»: предполагается включённый **AMP + CAB** (или SnapTone + IR).

## Когда CAB выключают

Если выход в **INPUT** обычного гитарного комбика — реальный динамик комбика уже кабинет → часто **CAB OFF**, иначе «кабинет дважды».

См.: [сценарии подключения](../manual-ru/08-soft-i-scenarii.md), [PA](./pa.md).

## FRFR vs наушники vs комбик

| Мониторинг | CAB / IR |
|------------|----------|
| FRFR / студийные мониторы | обычно ON |
| Наушники | обычно ON |
| Гитарный комбик в Input | часто CAB OFF |
| PA / FOH | обычно ON (как DI), режет звукорежиссёр |
