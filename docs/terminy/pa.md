# PA

**PA** (Public Address) — система озвучивания зала: микшер, усилители, фронтальные колонки (FOH).

В текстах проекта «войти в PA» = отдать [линейный / DI](./di.md)-сигнал с GP-180 звукорежиссёру (часто как DI), а не слушать только свой комбик.

Настройка патча при уходе в PA близка к логике [FRFR](./frfr.md): обычно **CAB ON**, уровни без клиппинга, без лишнего низа ([HPF](./hpf.md)).
