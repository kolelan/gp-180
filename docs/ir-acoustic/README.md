# Акустические IR как симулятор кабинета (CAB)

На GP-180 в слоте **CAB → User IR 1–20** можно загружать не только «гитарные 4×12», но и **импульсы акустических инструментов / тел / студийных съёмов** — как окраску корпуса для пьезо и акустики.

Это **не** NAM и не полная замена живому дереву: короткий IR (1024/2048 точек) даёт характер тела/микрофона, а не длинную комнату.

## Можно ли так делать?

**Да**, если файл совместим с Valeton Suite:

| Параметр | Значение на GP-180 |
|----------|-------------------|
| Слоты | User IR 1–20 |
| Длина | **1024** или **2048** точек |
| Частота | ориентир под движок **48 кГц** |
| Загрузка | Valeton Suite → управление IR → синхронизация с устройством |

Встроенные акустические симы `AC` / `OM` / `Bird` / `JUMBO` / `GA` — заводской аналог идеи; User IR даёт **свой** профиль.

## Главный сценарий

**Пьезо + IR инструмента в CAB + примочки** (AMP/NAM не обязательны):

→ подробный разбор: [`scenarii/piezo-body-ir-plus-fx.md`](./scenarii/piezo-body-ir-plus-fx.md)

Кратко:

```
пьезо → COMP → (AMP OFF) → CAB User IR (тело, напр. D-35)
      → EQ → chorus/delay/reverb
```

## Типичная цепь (с лёгким AC Pre)

```
NR → PRE (COMP / Saturate) → AMP AC Pre (Gain низкий)
→ CAB = User IR (акустический профиль)
→ EQ → (короткий DLY/RVB) → VOL
```

- Хайгейн AMP / SnapTone обычно **OFF**.
- Если IR уже «с микрофоном», не дублируйте длинный Hall/Shimmer.
- **Low Cut / High Cut** в CAB и патчевый EQ обязательны (quack, howl, микс).

## Ограничения

| Момент | Практика |
|--------|----------|
| Короткая длина IR | Хорошо для тела/«колонки»; плохо как реверб-комната |
| Пьезо + IR | Полезно; часто важнее связка **AC Pre + EQ** |
| Мандолина ≠ гитара | Гитарный acoustic IR на мандолине может звучать чуждо |
| DSP | Precision CAB + тяжёлая цепь — следите за нагрузкой |
| Лицензии | Кладите только IR, которые **разрешено** хранить и распространять |

## Какие IR выбирать (мандолина / акустика)

Частый совет: **студийные mic IR без жёсткого V30-scoopа**.

Расшифровка терминов: [`terminy-mic-ir-v30-scoop.md`](./terminy-mic-ir-v30-scoop.md).

Кратко: V30 + mid-scoop = рок/метал-кабинет; для акустики и мандолины лучше открытые 1×12 / transparent mic IR или **body IR** инструмента.
