# Практика EQ на GP-180

## Где стоит EQ в цепи

Рекомендуемый порядок для [FRFR](../terminy/frfr.md)/[PA](../terminy/pa.md):


`… AMP/SnapTone → CAB → EQ → MOD → DLY → RVB …`

EQ после кабинета ближе к тому, что услышит зал.

## Модули

| Модуль | Когда |
|--------|--------|
| Guitar EQ 1 | Мандолина/ритм в бэнде (есть 800 и 1.6k) |
| Guitar EQ 2 | Борьба с 6k и сибилянтной резкостью |
| Bass EQ 1 | Если через GP-180 идёт бас |
| Mess EQ | Metal V-shape; для мандолины редко |
| Global EQ | Компенсация комнаты/мониторов, не персональный тон |
| CAB Low/High Cut | Первый «грубый» EQ всегда |
| AMP/SnapTone Bass/Mid/Treble/Presence | Форма до кабинета |

## Порядок правок (KISS)

1. [Low Cut](../terminy/low-cut-high-cut.md) в CAB.
2. Tone/Treble/Presence на amp.
3. Патч EQ ±2…4 дБ по полосам.
4. Уровень Volume патча под PA.
5. Только потом Global EQ.
