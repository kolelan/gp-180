# Вокал

Важные зоны вокала: ~200–400 (тепло), **1–5 кГц** (разборчивость), 5–8 кГц (воздух/сы).

Мандокастер с магнитным датчиком легко маскирует слова на 2–5 кГц.

**Делать:**

- при конфликте −EQ 4k / −Presence на мандо, не «ярчить» вокальный монитор бесконечно;
- короче реверб на инструменте, чем на вокале;
- в припеве CTRL: −Mix Delay/Reverb на мандо или чуть Volume вниз.

**Не делать:** жёсткий Mess scoop + яркий Presence — получите «дырку» и крик сверху.
