# Ударные

| Элемент | Конфликт с мандолиной |
|---------|------------------------|
| Бочка | [низ](../terminy/niz-v-mikse.md) — лечится [HPF](../terminy/hpf.md) на струнном |
| Малый | 200–400 и щелчок 2–4k — не забивать тем же пиком на мандо |
| Хэты/тарелки | 6–12 kHz — [High Cut](../terminy/low-cut-high-cut.md) / Guitar EQ2 −6k |

На живой сцене без постоянного звукорежиссёра: **Room Mix низкий**, delay sync в темп, без длинного Shimmer.

Проверка: играйте чоп вместе с хэтом — если каша, режьте 6k и Mix reverba.
