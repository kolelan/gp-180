# Эквалайзер GP-180 в составе бэнда

Цель: посадить **мандолину / гитару** так, чтобы вместе жили **бас, ритм-гитара, вокал, ударные**.

Для стилей **фолк / регги / dark americana** с электрогитарой в составе см. также готовые clean/edge-патчи и таблицы микса: [../presets/elektro-ansambl/](../presets/elektro-ansambl/).

## Короткий алгоритм на репетиции

1. Соло ваш патч → уберите гул ([HPF](../terminy/hpf.md)) и резкость (3–6 кГц).
2. Добавьте **бас + бочку** → ещё [низ](../terminy/niz-v-mikse.md) мандолины, если мажет.
3. Добавьте **ритм-гитару** → разведите середины (кому 800 Гц, кому 1.6–2.5 кГц).
4. Добавьте **вокал** → не маскуйте 2–5 кГц; при конфликте режьте гитару/мандо, не голос.
5. Добавьте **тарелки** → High Cut / −6 кГц на ярких пресетах.
6. Глобальный EQ площадки трогайте последним и минимально.
