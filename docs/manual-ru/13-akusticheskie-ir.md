# Акустические IR как CAB (User IR)

На GP-180 в **CAB → User IR** допустимы не только гитарные кабинеты, но и **импульсы акустических инструментов** (тело / студийный съём), если файл **1024 или 2048** точек и загружен через Valeton Suite.

## Основной сценарий

Своя гитара или мандолина с **пьезо** → **User IR инструмента** в CAB (например профиль в духе Martin D-35) → **примочки**.  
Симулятор усилителя и **NAM/SnapTone не обязательны**.

Подробно: [сценарий «пьезо + body IR + FX»](../ir-acoustic/scenarii/piezo-body-ir-plus-fx.md)  
Каталог файлов: [IR-акустика](../ir-acoustic/).

Краткая цепь:

`COMP → (AMP OFF) → CAB User IR → EQ → MOD/DLY/RVB`

Вариант с лёгким пре: `AC Pre → CAB User IR → EQ`

Ограничения: короткий IR ≠ длинный reverb; мандолине лучше свои профили; следите за лицензией файла и DSP load.

Для электро-кабинетов рядом с акустикой/мандолиной: открытые mic IR **без жёсткого V30-scoopа** — [термины](../ir-acoustic/terminy-mic-ir-v30-scoop.md).
