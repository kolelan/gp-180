# ПО и сценарии подключения

## Valeton Suite

Бесплатное приложение: Windows / Mac / iOS / Android.

Возможности:

- правка патчей и параметров;
- пакетная загрузка **SnapTone** и **IR**;
- обновление прошивки;
- удобный drag-and-drop цепи.

ПК: USB. Мобильные: BT (и/или USB в зависимости от платформы).

### Windows 11: импорт NAM/IR и прошивка

Если устройство «не видится» для передачи файлов / прошивки — переключите драйвер GP-180 на универсальный **USB Audio Device** (см. [официальный Win11 notice](https://www.valeton.net/win11-technical-notice/)). Не используйте «чужие» MIDI/ASIO-привязки для этого workflow.

## Сценарии

### Полный диапазон (FRFR / наушники / PA / мониторы студии)

AMP + CAB (или SnapTone + IR) включены. Global EQ под акустику комнаты.

### Комбик в INPUT

Часто **CAB OFF** (кабинет у усилителя реальный). AMP — по вкусу или только пре-эффекты + DST. Уровень OUT не клиппить вход комбика.

### FX Loop комбика (4CM / return)

GP-180 в Return или в loop: типично кабинет сима выключают; используют delay/reverb/mod после преампа комбика.

### GP-180 перед преампом (инструмент → GP-180 → Input amp)

Больше педальной логики: OD/wah/boost перед своим любимым комбиком; amp/cab внутри GP-180 часто выключены.

### Внешнее управление

EXP / доп. футсвитчи / MIDI через TRS jack и USB/BT.

### Аудиостудия и стрим

USB как интерфейс; наушники TRRS; при необходимости лупер и драм-машина для метронома/бэкграунда.
