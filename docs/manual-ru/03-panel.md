# Панель и разъёмы

## Органы управления (лицевая панель)

| # | Элемент | Назначение |
|---|---------|------------|
| 1 | LCD | Номер/имя патча и рабочая информация |
| 2 | Индикатор зарядки | Красный при зарядке |
| 3 | MASTER | Общая выходная громкость |
| 4 | PARA (+ кнопка) | На главном экране — смена патчей; в Edit удержание — смена порядка модулей |
| 5 | Quick Access | Быстрая правка параметров внизу экрана |
| 6 | BACK | Назад |
| 7 | GLOBAL | Глобальные настройки |
| 8 | BT | Вкл/выкл Bluetooth |
| 9 | MODE | Patch Mode ↔ Stomp Mode |
| 10 | SAVE | Сохранить / переименовать / копировать; удержание — quick-save |
| 11 | DRUM | Пуск драм-машины; удержание — экран драм-машины |
| 12 | EXP Indicator | Статус педали экспрессии |
| 13 | BT Indicator | Поиск (мигает) / связь (горит) |
| 14 | 3 футсвитча | Патчи, эффекты, Tap Tempo и др. |
| 15 | Module Button | Статус модулей; нажатие — переход к модулю; удержание — вкл/выкл |

## Разъёмы

| Элемент | Описание |
|---------|----------|
| EXP/FS 2 (MIDI OUT) | 6,35 мм TRS: внешний контроллер / MIDI OUT |
| EXP/FS 1 (MIDI IN) | 6,35 мм TRS: контроллер / MIDI IN |
| IN | 6,35 мм TS — гитара / мандолина / другой инструмент |
| OUT (L) | 6,35 мм TS — небалансный; для моно достаточно L |
| OUT (R) | 6,35 мм TRS — баланс/небаланс, mono/stereo |
| PHONES/MIC | 3,5 мм TRRS — наушники / гарнитура |
| USB Type-C | Данные, аудиоинтерфейс, питание/зарядка, reverse charge |
| POWER | Вкл/выкл / сброс |
| DC 9V | Питание / зарядка |
