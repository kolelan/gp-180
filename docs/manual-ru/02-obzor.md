# Обзор GP-180

GP-180 — цифровой мультиэффект нового поколения в форм-факторе, развитом от GP-100, с упором на цену/качество.

## Ключевые возможности

- Гибкая цепь: до **12** свободно перемещаемых модулей.
- Более **200** эффектов на HD-моделировании 2-го поколения.
- Частота дискретизации **48 кГц**.
- **SnapTone** — конвертация и загрузка **NAM** (Neural Amp Modeler): 50 заводских файлов, всего до **100**.
- Кабинетные **IR**: до **20** сторонних (1024 / 2048 точек).
- Дисплей высокого разрешения.
- Внешние футсвитчи / экспрессия, **MIDI**.
- Редактор Valeton Suite: Windows / Mac / iOS / Android.
- Режим **аудиоинтерфейса**.
- Встроенный **BT**.
- Наушники / гарнитура TRRS.
- Драм-машина: **100+** паттернов.
- Лупер до **180** с.
- До **200** патчей (100 заводских).

Для живых выступлений, репетиций, домашней записи и стриминга.
