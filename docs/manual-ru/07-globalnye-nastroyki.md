# Глобальные настройки

Кнопка **GLOBAL**. Основные разделы:

## Input / Output

Уровень входа, режим выхода, mono/stereo, уровни для наушников и линий. Для мандокастера с «горячим» датчиком при клиппинге входа снижайте Input или Volume на инструменте.

## USB Settings

Режим аудиоинтерфейса, маршрутизация. GP-180 может работать как интерфейс к ПК / телефону.

## Global EQ

Общий эквалайзер поверх всех патчей — удобен для площадки/мониторов. Для посадки в бэнд чаще правят **патчевый EQ** (см. [eq-mix](../eq-mix/)); Global EQ — только если вся система стабильно «темнее/ярче».

## TAP Settings

Поведение Tap Tempo, синхронизация модуляций/delay.

## EXP Calibrate

Калибровка выражения. После калибровки назначьте параметр (Wah Position, Volume, Whammy и т.д.).

## Footswitch

Patch/Stomp по умолчанию, поведение светодиодов.

## BT Settings

Bluetooth: редактор / аудио. Для мобильного Suite.

## Auto Save

Автосохранение правок (осторожно: можно перезаписать патч нечаянно).

## Display

Яркость, таймауты.

## EXP/FS

Тип контроллера на Jack 1/2 (EXP, FS, MIDI).

## Bypass Screen

Поведение байпаса.

## MIDI Settings

Канал, PC/CC поведение. Таблица CC — [09-midi.md](./09-midi.md).

## Factory Reset

Полный сброс. **Сотрёт пользовательские данные** — делайте бэкап патчей/SnapTone через Suite.

## About

Версия прошивки, серийный номер.
