# Начало работы

1. Подключите инструмент в **IN**, кабель с **OUT L** — в комбик (или в **FX Return**, если есть loop). Громкость комбика — низкая.
2. Крутилку **MASTER** поверните вниз, затем включите GP-180.
3. **Тюнер:** удерживайте футсвитчи **A и B**. Настраивайте, пока индикация станет зелёной по центру. Выход: любой футсвитч или BACK.
4. Выберите патч футсвитчами A/B или крутилкой PARA.
5. Поднимите громкость комбика и MASTER, играйте.

Для мандокастера удобно начинать с чистых/лёгких заводских патчей (Dark Clean, Foxy Clean, JC-подобные) или пресетов из [../presets/mandocaster/](../presets/mandocaster/).
