# Устранение неисправностей

| Симптом | Что проверить |
|---------|----------------|
| Нет звука | Кабели, MASTER, громкость патча, Mute тюнера, заряд <5%, правильный OUT |
| Только шум | Gate, Gain amp, экранирование инструмента, BT/помехи |
| Тихий звук | Input level, Volume модулей, CAB Volume, Global EQ |
| Клиппинг | Input, Gain цепочки, выход в интерфейс |
| Не грузятся NAM/IR (Win11) | Драйвер → USB Audio Device |
| Неверная экспрессия | Калибровка EXP, назначение параметра, включённый модуль Wah/Volume |
| Высокая DSP load | Отключить Precision CAB / лишние FX / один из AMP|SnapTone |

Сервис: **service@valeton.net**.
