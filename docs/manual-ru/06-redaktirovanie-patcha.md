# Редактирование патча

## Edit Screen

Вход: Module Button или редактирование через Suite.

На экране — цепь эффектов. Цветные блоки = модули.

Порядок по умолчанию:

`NR → PRE → WAH → DST → N→S → AMP → CAB → EQ → MOD → DLY → RVB → VOL`

### DSP Load

Следите за индикатором нагрузки. Тяжёлые комбинации (SnapTone + Precision CAB + несколько слоёв FX) могут превысить бюджет DSP — упростите цепь или Precision.

### Смена порядка модулей

Удерживайте **PARA** и перетаскивайте / меняйте порядок модулей (на устройстве — через удержание PARA в Edit; удобнее в Suite drag-and-drop).

### CTRL-модули

В Edit назначьте до нескольких модулей на футсвитчи A/B/C (CTRL) для живого вкл/выкл (соло-буст, delay, октавер и т.д.).

### Patch Settings

Громкость патча, BPM, привязки Quick Access Knob к параметрам (Gain amp, Mix delay и пр.).

## Save Screen

SAVE → имя / слот / копирование → SAVE ещё раз. Удержание SAVE — быстрое сохранение.

User-патчи: слоты сверх заводских (всего 200, из них 100 factory).
