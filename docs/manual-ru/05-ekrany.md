# Экраны

## Главный экран

Два режима футсвитчей (LED-кольца):

- **Patch Mode** (синий): смена патчей A/B; на экране номер и имя патча.
- **Stomp Mode** (зелёный/красный): A/B включают/выключают назначенные модули.

Переключение: удержание B, кнопка MODE или Global Settings.

На главном экране также: уровни I/O, Volume/BPM и пользовательские параметры Quick Access, статус EXP/FS, драм-машины, reverse USB power, батарея.

### Футсвитч C (CTRL / TAP)

По умолчанию **CTRL** (как Stomp для назначенных модулей). Удержание C переключает **CTRL ↔ TAP**.

- **TAP:** BPM 40–300 по частоте нажатий; кольцо мигает в темпе.

### Низкий заряд

- <20% — предупреждение (можно закрыть PARA).
- <5% — предупреждение нельзя закрыть, **звук отключается**. Подключите зарядку.

## Тюнер

Удержание A+B. Хроматический тюнер, шаблоны строев, A4 = 435–445 Гц (по умолчанию 440).

Режимы: **Mute** / **DSP Bypass** / **Ana. Bypass** (аппаратный true bypass, mono; для стерео предпочтительнее DSP Bypass).

Шаблоны: Off, Standard, Drop D/C/A, Bass 4/5, Open D/E/G.

> Удержание A+B >2 с открывает **Looper**.

## Лупер

До **180** с записи/овердаба; auto-record и синхрон с драм-машиной.

| Действие | Футсвитч |
|----------|----------|
| Запись / воспроизведение / овердаб | A |
| Стоп | B |
| Очистить | удержание B |
| Tap Tempo | C |

Параметры: Rec VOL, Pre/Post в цепи, Play VOL, Drum Sync, Auto REC.

- **Pre:** сухой mono без эффектов → можно менять патч.
- **Post:** мокрый stereo; память ×2 (макс. ~90 с).

Контент лупера системой **не сохраняется**.

## Драм-машина

DRUM — вкл/выкл. Удержание DRUM — экран стилей/параметров. 100+ паттернов. BPM связан с TAP патча при синхронизации.
