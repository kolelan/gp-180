# Как импортировать NAM → SnapTone

## Что нужно

1. Файл модели **`.nam`** (лучше проверенный A/Standard capture под ваш интерфейс).
2. **Valeton Suite** с сайта Valeton.
3. GP-180 по **USB** (на ПК) или BT (моб., если Suite поддерживает файловый обмен).

## Шаги

1. Установите Suite, подключите GP-180, дождитесь связи.
2. Откройте раздел управления **SnapTone / N→S**.
3. Выберите **пустой** слот (51–100 для пользовательских; заводские 1–50 можно перезаписывать только осознанно — сделайте бэкап).
4. Импортируйте `.nam` — Suite конвертирует в SnapTone и синхронизирует с устройством.
5. На GP-180: патч → модуль **N→S** → выберите слот → крутите Gain/EQ/Presence.
6. Решите, нужен ли **CAB/User IR** (см. README раздела).

## Советы по качеству

- Модели **amp-only** + отдельный IR обычно гибче, чем «amp+cab» one-shot.
- Очень «горячие» NAM: снизьте Input/Gain; поставьте **NR Gate 3** перед N→S.
- Если импорт на **Windows 11** падает — переключите драйвер устройства на **USB Audio Device** ([инструкция Valeton](https://www.valeton.net/win11-technical-notice/)).
- Часть пользователей сообщает, что на Win11 удобнее грузить с iPad/Android — имейте план B.

## IR (кабинет)

В Suite: раздел **User IR** → слоты 1–20 → WAV IR 1024 или 2048 точек → на устройстве CAB → User IR N.

Для мандолины чаще работают прозрачные 1×12 / студийные mic IR без жёсткого V30-скoopа.
