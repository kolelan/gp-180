# Мандолина Vox chime

## Для чего
Яркий «колокольчик» AC30: кельт, инди, арпеджио поверх ритм-гитары.

## Когда
- Да: нужен воздух и чёткий транзит.
- Нет: уже крикливая PA / хриплый вокал без HPF.

## Источник
Заводской **Foxy CL** (часто с 2×12) или Match 35 CL.

## Цепь
`NR → PRE(COMP4 мягко) → N→S Foxy CL → AMP OFF → CAB OFF (если cab в модели) или Foxy 2x12 → EQ2 → MOD(G-Chorus Depth низкий) → DLY Digital короткий → RVB Plate Mix низкий`

## Старт

| Блок | Параметр | Значение |
|------|----------|----------|
| N→S | Gain | 10–30 |
| N→S | Treble | 30–40 |
| EQ2 | 6 kHz | −3…−6 (убрать шип) |
| EQ2 | 1 kHz | +1…+2 |
| Chorus | Depth | 15–30 |
| Plate | Mix | 10–20 |

## Микс
Chime конкурирует с хэтами и сибилянтами вокала — обязательно High Cut / −6k.
