# Ambient: Freeze + Shimmer

## Для чего
Дрон/подложка под вокал или импров мандолины на сцене с лупером.

## Цепь
`NR → N→S Dark CL → CAB Twin → EQ → MOD Freeze → DLY Vintage Rack → RVB Shimmer`

## Параметры
- Freeze: Activate на CTRL; Attack/Release плавные.
- Shimmer Mix: 10–20 в бэнде, 25–40 в соло-эмбиенте.
- Looper Post: записывать уже «мокрый» слой, Pre — чтобы менять патчи поверх дрона.

## Когда нет
Плотный быстрый блюграсс с акустикой — реверб/шиммер размоет чоп.
