# Ритм-гитара рядом с мандолиной

## Для чего
Патч для второго музыканта (ритм-электро) в том же бэнде, где уже играет мандокастер — чтобы два инструмента не сидели в одной частотной яме.

## Источник
AMP **UK 800** / **Flagman 1** с умеренным Gain **или** SnapTone UK Force с низким Gain.

## Цепь
`NR → DST Green OD (mid boost) → AMP UK… → CAB UK 2x12 → EQ Mess или Guitar EQ1 → MOD off → DLY Slapback еле → RVB Room`

## EQ-идея
- Мандолина: упор 1.5–3 кГц, мало низа.
- Ритм: чуть больше 200–500 Гц тела, **вырезать** 1.6–2.5 кГц (−1…−3), чтобы отдать слот мандолине и вокалу.
- Оба: HPF ~80–100 Гц.

## Правило
В одном бэнде **не** делайте оба пресета ярко-scoopy V-shape — потеряете середину и «склеитесь» с тарелками.
