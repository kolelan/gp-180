# Рецепты SnapTone / NAM

Рецепт = готовый сценарий: зачем, какая модель, цепь, стартовые параметры, когда не использовать.

## Как добавить новый рецепт

1. Скопируйте [`_shablon.md`](./_shablon.md) → `NN-korotkoe-imya.md`.
2. Заполните все поля.
3. Добавьте ссылку в таблицу ниже.
