# Чистая мандолина (Fender SnapTone)

## Для чего
Основной clean для мандокастера: акценты, арпеджио, баллады рядом с акустической гитарой и вокалом.

## Когда использовать / когда нет
- Да: народ, кантри, джаз-мандолина, запись DI в FRFR/наушники.
- Нет: тяжёлый рок-чоп с овердрайвом, замена басовой партии.

## Источник модели
- Заводской: **Dark CL** или **Dark DLX** / **Dark PRI**
- Альтернатива NAM: любой качественный Twin/Deluxe/Princeton amp-only + прозрачный 1×12 IR  
(без жёсткого V30-scoopа — см. [термины](../../ir-acoustic/terminy-mic-ir-v30-scoop.md)).


## Цепь
`NR(Gate3) → PRE(COMP мягко) → N→S ON → AMP OFF → CAB ON (Twin 2x12 или User IR) → EQ(Guitar EQ1) → MOD(Detune еле-еле, опц.) → DLY(Slapback низкий) → RVB(Room/Spring) → VOL`

## Стартовые параметры

| Блок | Параметр | Значение (ориентир 0–100) |
|------|----------|---------------------------|
| Gate3 | Threshold | 25–40 |
| COMP | Sustain | 25–40 |
| N→S | Gain | 15–35 |
| N→S | Presence | 30–45 |
| N→S | Bass | 40–50 |
| N→S | Middle | 50–55 |
| N→S | Treble | 35–45 |
| CAB | Low Cut | ~80–120 Hz |
| CAB | High Cut | ~8–10 kHz |
| Guitar EQ1 | 125 Hz | −2…−4 |
| Guitar EQ1 | 400 Hz | −1…−3 |
| Guitar EQ1 | 800 Hz | 0…+2 |
| Guitar EQ1 | 1.6 kHz | +1…+3 |
| Guitar EQ1 | 4 kHz | −2…−4 |
| Slapback | Mix | 10–20 |
| Room | Mix | 15–25 |

## Варианты
1. Без кабинета SnapTone Dark DLX (уже с cab) → CAB OFF.
2. Больше «перьев»: не поднимать Treble/4k, лучше слегка 1.6k.

## Микс
Оставляйте ~80–200 Гц басу и кику; не залезайте в 2–5 кГц вокала жёсткой яркой полкой.

## CTRL / EXP
CTRL: Detune или короткий Delay. EXP: VOL для свэллов.
