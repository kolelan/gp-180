# Лёгкий драйв соло мандолины

## Для чего
Соло поверх бэнда: чуть сатурации, сустейн, читаемость без «metal wall».

## Когда
- Да: припев соло, фьюжн.
- Нет: ритм-чоп в голубой траве — останетесь грязнее флэттопа.

## Источник
**Band OD** / **ToneK OD** / **Match 35 OD** с низким Gain; либо Pedal SnapTone **KOT OD** / **J-RAY OD** + чистый amp Dark Twin (AMP) и CAB.

## Цепь (педаль SnapTone + clean amp)
`NR → PRE(Micro Boost опц. на CTRL) → N→S = KOT OD (Gain низкий) → AMP Dark Twin → CAB Twin 2x12 → EQ1 → DLY Tape → RVB Hall Mix низкий`

## Старт

| Блок | Параметр | Значение |
|------|----------|----------|
| N→S (KOT) | Gain | 20–35 |
| AMP | Gain | 20–40 |
| EQ1 | 800 Hz | +2…+4 (прорезь) |
| EQ1 | 125 Hz | −3 |
| Delay | Mix | 15–25, Feedback низкий |

## CTRL
Micro Boost или поднять Mix Delay только на соло.
