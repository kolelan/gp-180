# Источники NAM-моделей

Импортируйте только то, на что у вас есть право (лицензия автора capture / магазин / собственный замер).

## Каталоги и сообщества

| Источник | Что искать | Комментарий |
|----------|------------|-------------|
| [Tone3000](https://www.tone3000.com/) | NAM packs amp/pedal | Большой каталог, фильтры по amp |
| [Neural Amp Modeler](https://www.neuralampmodeler.com/) / экосистема NAM | Доки, тулзы | Официальная платформа формата |
| [GuitarML](https://guitarml.com/) | Модели и обучение | Открытая экосистема |
| Тоне-пакеты производителей (Revv, Fortin и др.) | Иногда `.nam` | Читайте EULA — не все разрешают конвертацию в железо |
| Собственный capture | Ваш комбик/педаль | Максимально «ваш» звук |

## Что качать для мандокастера

Ищите по ключевым словам:

- `blackface`, `twin`, `deluxe reverb`, `princeton` — открытый american clean;
- `ac30 top boost clean`, `matchless` — британский chime без каши;
- `jazz chorus`, `solid state clean` — ровный clean без сатурации;
- `edge of breakup`, `low gain` — едва тронутый преамп;
- избегать сразу: `5150 lead`, `rectifier modern`, `ultra high gain` — слишком плотно для чопа мандолины в акустическом бэнде.

## Педальные NAM

SnapTone на GP-180 включает и педальные captures (Rat, OCD, KoT и т.д. — см. заводской список). Их можно ставить **вместо** DST или вместе с чистым AMP — но не дублируйте один и тот же тип дисторшена дважды без цели.

## Чеклист перед импортом

- [ ] Лицензия позволяет использование / конвертацию
- [ ] Понятно: amp-only или amp+cab
- [ ] Есть запасной слот и бэкап
- [ ] Под вашу сцену есть IR (если модель без кабинета)
