# MOD — модуляция

После EQ. Chorus/flanger/phaser/trem/detune/swell/freeze.

## Эффекты в модуле

- [G-Chorus](./g-chorus/) — Chorus: Винтажный ensemble-chorus конца 70-х.
- [C-Chorus](./c-chorus/) — Chorus: 4-режимный stereo chorus.
- [B-Chorus](./b-chorus/) — Chorus/Vib: Вариант «фиолетового» chorus.
- [Jet](./jet/) — Flanger: Классический флангер.
- [B-Jet](./b-jet/) — Flanger: Флангер для баса.
- [V-Roto](./v-roto/) — Vibrato: BBD blue vibrato.
- [Vibrato](./vibrato/) — Vibrato: Классическое вибро.
- [O-Phase](./o-phase/) — Phaser: Phase 90*-стиль.
- [Vibe](./vibe/) — Phaser: Uni-Vibe: Chorus/Vibrato.
- [O-Trem](./o-trem/) — Tremolo: Opto trem (Demeter*).
- [Sine Trem](./sine-trem/) — Tremolo: Синусная форма.
- [Triangle Trem](./triangle-trem/) — Tremolo: Треугольная форма.
- [Bias Trem](./bias-trem/) — Tremolo: Bias-тремоло + Bias параметр.
- [Detune](./detune/) — Pitch: Лёгкий детюн = chorus-like.
- [Auto Swell](./auto-swell/) — Special: Авто-свэлл как скрипка.
- [Hold](./hold/) — Special: Freeze короткого фрагмента (Activate→EXP/CTRL).
- [Freeze](./freeze/) — Special: Заморозка звука с Attack/Release.

---

Всего эффектов: **17**.
