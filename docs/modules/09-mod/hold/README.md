# Hold

**Тип:** Special

## Описание

Freeze короткого фрагмента (Activate→EXP/CTRL).

## Параметры

- Volume
- Activate

## Варианты использования

### Live ambient layer

_Настройте на слух под площадку и инструмент._

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
