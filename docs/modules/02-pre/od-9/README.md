# OD 9

**Тип:** Overdrive

## Описание

TS9-стиль: прозрачный буст преампа, кранч-ритм.

## Параметры

- Gain
- Tone
- Volume

## Варианты использования

### Перед Marshall/UK

Gain 30–45, Tone 45–55.

### Мандолина в миксе

Gain низкий как mid-boost ~700–900 Гц.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
