# COMP4

**Тип:** Comp

## Описание

Студийный 4-рычажный компрессор (Keeley® C4*). Иерархия и яркий верх.

## Параметры

- Sustain
- Attack
- Volume
- Clipping

## Варианты использования

### Соло поверх микса

Быстрый Attack, Sustain 50–70.

### Бас / низ мандолины

Мягкий Attack, Clipping не задирать.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
