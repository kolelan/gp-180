# T-Wah

**Тип:** Filter

## Описание

Touch wah / envelope filter: реагирует на силу игры.

## Параметры

- Sens
- Range
- Q
- Mix
- Mode Guitar/Bass

## Варианты использования

### Фанк

Sens под атаку, Mode Guitar.

### Мандолина chop

Sens умеренный, Mix 40–60 — не размывать атаку.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
