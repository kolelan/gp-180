# Saturate

**Тип:** Special

## Описание

Ленточная сатурация: аналоговая теплота.

## Параметры

- Mix
- Volume
- High Cut

## Варианты использования

### Склеить цифровой clean

Mix 20–40, High Cut чуть вниз.

### Мандокастер warmer

Лёгкая сатурация вместо жёсткого OD.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
