# AC Sim

**Тип:** Simulator

## Описание

Симулятор акустики (Standard/Jumbo/Enhanced/Piezo).

## Параметры

- Body
- Top
- Volume
- Mode

## Варианты использования

### Электрика → акустика

Mode Standard/Jumbo.

### Мандолина acoustic-ish

Body умеренно, Top не задирать — магниты и так яркие.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
