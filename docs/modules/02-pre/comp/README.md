# COMP

**Тип:** Comp

## Описание

Компрессор в духе Ross™: мягкое натуральное сжатие.

## Параметры

- Sustain
- Volume

## Варианты использования

### Мандокастер — выровнять атаки

Sustain 35–50, Volume чуть выше unity.

### Чистый кантри-ритм

Sustain умеренный, сохранить динамику.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
