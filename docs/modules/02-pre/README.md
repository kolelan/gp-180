# PRE — Pre-Effects (пре-эффекты)

Перед вау и дисторшеном: компрессия, бусты, фильтры, питч, симуляторы.

## Эффекты в модуле

- [COMP](./comp/) — Comp: Компрессор в духе Ross™: мягкое натуральное сжатие.
- [COMP4](./comp4/) — Comp: Студийный 4-рычажный компрессор (Keeley® C4*).
- [Micro Boost](./micro-boost/) — Boost: Чистый буст до +20 дБ без смены характера (MXR® Micro Amp*).
- [B-Boost](./b-boost/) — Boost: Плотный кремовый буст/овердрайв (Xotic® BB Preamp*).
- [14 Boost](./14-boost/) — Boost: Агрессивный tight-буст (Fortin® Grind*): +20 дБ, низкий шум.
- [Boost](./boost/) — Boost: EP-style буст: сильный низ, яркий верх (Xotic® EP Booster*).
- [OD 9](./od-9/) — Overdrive: TS9-стиль: прозрачный буст преампа, кранч-ритм.
- [Yellow OD](./yellow-od/) — Overdrive: Асимметричный овердрайв 70-х: звук близок к исходному.
- [Penesas](./penesas/) — Overdrive: Klon® Centaur*-стиль: amp-in-a-box; Gain=0 → clean boost.
- [Super OD](./super-od/) — Overdrive: Тёплый асимметричный овердрайв.
- [Blues OD](./blues-od/) — Overdrive: Выразительный OD от тёплого до открытого дисторшена.
- [T-Wah](./t-wah/) — Filter: Touch wah / envelope filter: реагирует на силу игры.
- [A-Wah](./a-wah/) — Auto Filter: Авто-вау с заданной скоростью.
- [Step Filter](./step-filter/) — Filter: 4-шаговый автофильтр — синтоподобные тембры.
- [OCTA](./octa/) — Pitch: Полифонический октавер.
- [Pitch](./pitch/) — Pitch: Полифонический питч-шифтер / гармонайзер.
- [P-Bend](./p-bend/) — Pitch: Полифонический bend/harmony с диапазоном.
- [Hammy](./hammy/) — Pitch: Whammy®*-стиль (моно).
- [Ring Mod](./ring-mod/) — Special: Кольцевой модулятор: колокола, металлические тембры.
- [Saturate](./saturate/) — Special: Ленточная сатурация: аналоговая теплота.
- [AC Sim](./ac-sim/) — Simulator: Симулятор акустики (Standard/Jumbo/Enhanced/Piezo).
- [H to S](./h-to-s/) — Simulator: Симулятор: хамбакер → сингл (ST bridge).
- [S to H](./s-to-h/) — Simulator: Симулятор: сингл → хамбакер (LP bridge).

---

Всего эффектов: **23**.
