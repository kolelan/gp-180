# B-Boost

**Тип:** Boost

## Описание

Плотный кремовый буст/овердрайв (Xotic® BB Preamp*).

## Параметры

- Gain
- Volume
- Bass
- Treble

## Варианты использования

### Тёплый буст под clean amp

Gain низкий, Bass чуть вверх.

### Густой ритм

Gain средний, Treble слегка вниз.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
