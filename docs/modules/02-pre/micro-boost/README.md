# Micro Boost

**Тип:** Boost

## Описание

Чистый буст до +20 дБ без смены характера (MXR® Micro Amp*).

## Параметры

- Gain

## Варианты использования

### Push amp into OD

Gain 30–50 перед ламповым AMP.

### Соло-буст

CTRL на Micro Boost, Gain 40–60.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
