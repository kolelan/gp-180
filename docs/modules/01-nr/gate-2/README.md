# Gate 2

**Тип:** Gate

## Описание

Гибкий шумодав с атакой и релизом.

## Параметры

- Threshold
- Attack — как быстро открывается
- Release — как быстро гасит шум

## Варианты использования

### Метал с палм-мьютом

Attack короткий, Release средний.

### Динамичная чистая игра

Attack длиннее, Threshold мягче.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
