# Gate 1

**Тип:** Gate

## Описание

Простой шумодав: сигнал ниже порога глушится.

## Параметры

- Threshold — порог срабатывания
- Decay — скорость затухания шума

## Варианты использования

### Ритм с дисторшеном

Threshold 40–55, Decay короткий — чёткие паузы между риффами.

### Лид с сустейном

Threshold ниже, Decay длиннее — не обрезать хвост ноты.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
