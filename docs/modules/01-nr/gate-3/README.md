# Gate 3

**Тип:** Gate

## Описание

Inverse Expander: убирает шум, сохраняя сустейн и динамику.

## Параметры

- Threshold
- Attack
- Release
- Hold — удержание открытого состояния

## Варианты использования

### High-gain SnapTone/NAM

Gate 3 перед N→S: Threshold по шуму усилителя, Hold 20–40.

### Акустика / мандолина

Мягкий Threshold; не зажимать тихие атаки.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
