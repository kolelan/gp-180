# Gate 3

**Тип:** Gate (шумодав)  
**Модуль:** [NR](../)

## Описание

Оригинальный алгоритм **Inverse Expander** (по Effect List GP-180): точечно убирает шум **до** усиления, при этом лучше, чем простой hard-gate, **сохраняет сустейн и динамику**.

Стоит первым в цепи по умолчанию. Настраивайте по **уровню шума без игры** (помехи, гул amp/SnapTone, BT) и по тому, как гаснет реальная нота.

Карточка опирается на параметр-лист устройства и описание в англ. Effect List (`Threshold` / `Attack` / `Release` / `Hold`). Параметр **Ratio** есть в UI Gate 3 (шкала 0–100); в кратком Effect List PDF он может не быть назван отдельно — ниже он разобран как жёсткость расширения вниз.

---

## Параметры и единицы

На экране GP-180 цифры — **шкала процессора**, не «открытые» dB/мс с подписью. По диапазонам и смыслу:

| Параметр | Диапазон UI | Единицы (практический смысл) | На что влияет |
|----------|-------------|------------------------------|---------------|
| **Threshold** | 0–100 | Условная шкала порога (не подписанные dB) | **Выше** = гейт срабатывает «раньше» / агрессивнее: тихий сигнал и хвосты нот чаще уходят в глушение. **Ниже** = мягче, больше слышен сустейн и тихие ноты, но и больше шума в паузах |
| **Ratio** | 0–100 | Условная «жёсткость» расширения вниз | **Ниже** ≈ мягкое приглушение шума (expander). **Выше** ≈ ближе к жёсткому gate: паузы суше, хвосты режутся сильнее |
| **Attack** | 1–500 | По сути **миллисекунды (ms)** | Как быстро гейт **открывается** / начинает пропускать сигнал, когда уровень перешёл порог. Короткий Attack — атака медиатора целее; длинный — смягчает фронт («проглатывает» щипок) |
| **Release** | 10–10000 | По сути **миллисекунды (ms)** (до ~10 с) | Как долго **затухает** глушение после того, как сигнал упал ниже порога. Короткий — резкая тишина (чоп/регги); длинный — хвост тремоло/лида не обрывается |
| **Hold** | 0–1000 | По сути **миллисекунды (ms)** | Сколько держать **открытое** состояние после падения под порог **до старта Release**. Больше Hold = меньше «дрожания» гейта на тремоло и неоднородной динамике |

> Если на вашей прошивке у Attack/Release/Hold видна явная подпись **ms** — ориентируйтесь на неё; логика та же.

### Как слушать порог

1. Не играйте → слышите шум паузы.  
2. Поднимайте **Threshold**, пока шум в паузе не уйдёт / не станет приемлемым.  
3. Сыграйте тихую длинную ноту / тремоло → если хвост «рубится», Threshold или Ratio ↓, либо Release/Hold ↑.

## Примеры настроек (ориентиры)

Значения — стартовые; крутите Threshold по своей сцене и уровню Input.

### 1. Folk / чистый лид (мандокастер, сустейн важнее тишины)

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| Threshold | **20–30** | Не душить тихие фразы |
| Ratio | **35–50** | Мягкий expander, не hard-chop |
| Attack | **5–20** | Сохранить щипок |
| Release | **180–350** | Хвост ноты живой |
| Hold | **40–100** | Тремоло не «дрожит» гейтом |

См. пресет: [Folk Clean](../../../presets/mandocaster/01-folk-clean.md).

### 2. Блюграсс-чоп / регги-skank (паузы важнее сустейна)

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| Threshold | **40–55** | Чёткая тишина между ударами |
| Ratio | **55–75** | Сухие паузы |
| Attack | **1–10** | Щелчок chick/чопа сразу |
| Release | **60–150** | Быстро в тишину |
| Hold | **10–40** | Короткие удары, без длинного «открыто» |

### 3. High-gain / шумный SnapTone·NAM

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| Threshold | **по шуму amp** (часто 35–55) | Убрать шипение модели |
| Ratio | **50–70** | Шум не «сочит» в паузах |
| Attack | **3–15** | Рифф не опаздывает |
| Release | **100–250** | |
| Hold | **20–50** | Как в старых заметках Hold 20–40 |

Ставьте Gate 3 **перед** N→S.

### 4. Ambient / длинные ноты / Freeze-pad

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| Threshold | **15–28** | Не резать swell |
| Ratio | **25–45** | Очень мягко |
| Attack | **10–40** | Можно чуть смягчить атаку |
| Release | **400–1200** | Длинный хвост |
| Hold | **80–200** | Стабильный слой |

### 5. Перекличка соло / плотные паузы без чопа

| Параметр | Ориентир | Зачем |
|----------|----------|--------|
| Threshold | **28–40** | Тишина между репликами |
| Ratio | **45–60** | |
| Attack | **5–15** | |
| Release | **120–220** | |
| Hold | **30–70** | |

## Gate 3 vs Gate 1 / Gate 2

| | Gate 1 | Gate 2 | Gate 3 |
|--|--------|--------|--------|
| Характер | Классический «Decimator-like» | Гибкий Attack/Release | Inverse Expander — шум ↓, сустейн живее |
| Когда | Просто и жёстко | Чоп / металл-паузы | Clean, мандолина, NAM с шипением, когда hard-gate «рубит» фразы |

---

## Варианты использования (кратко)

- **Мандолина / акустика:** мягкий Threshold + средний Release/Hold — иначе срежется сустейн тремоло.  
- **Чоп / skank:** Threshold и Ratio выше, Release короче.  
- **Шумный amp:** Threshold по паузе без игры; не лечите шип EQ, пока гейт не настроен.

---

## Заметки

- Имена брендов в Effect List (ISP® Decimator™ и т.п.) — только для описания характера **других** гейтов; Gate 3 — оригинальный алгоритм Valeton.  
- DSP: NR лёгкий; узкое место обычно SnapTone + Precision CAB + тяжёлые FX.  
- Англ. мануал: *Threshold — trigger level; Attack — how soon the gate starts to process; Release — noise fade-out after level drops below threshold; Hold — hold time for the previous state.*
