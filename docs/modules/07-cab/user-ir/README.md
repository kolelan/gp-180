# User IR 1–20

**Тип:** User IR

## Описание

Сторонние IR 1024/2048 точек.

## Параметры

- Volume
- Low Cut
- High Cut
- Precision

## Варианты использования

### Свой кабинет

Загрузка через Valeton Suite.

### Мандолина

IR открытых/прозрачных 1x12 или студийных DI-friendly.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
