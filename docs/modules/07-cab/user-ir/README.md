# User IR 1–20

**Тип:** User IR

## Описание

Сторонние IR 1024/2048 точек.

## Параметры

- Volume
- Low Cut
- High Cut
- Precision

## Варианты использования

### Мандолина / акустическая гитара

IR открытых тел, OM/малых корпусов, студийных DI-friendly съёмов — через **User IR**.  
Инструкция и папки профилей: [../../ir-acoustic/](../../ir-acoustic/).

### Свой кабинет (электро)

Классические cab IR 1×12 / 4×12 — загрузка через Valeton Suite.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
