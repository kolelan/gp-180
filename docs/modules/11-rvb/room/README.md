# Room

**Тип:** Reverb  
**Модуль:** [RVB](../)

## Описание

**Room** — реверб «небольшой комнаты»: короткий, относительно **плотный** хвост, ощущение близости. Не Hall (зал) и не Plate (студийная «пластина»).

На GP-180 в цепи обычно **после delay** (Slapback / Digital): сначала чёткий «шлёпок», потом мягкое пространство. Не путать с [Slapback](../../10-dly/slapback/) — тот **delay**, не reverb.

Параметр хвоста при байпасе на UI: **Trail** (в разговоре иногда «Tail»).

---

## Параметры

| Параметр | Единицы / тип | На что влияет |
|----------|---------------|----------------|
| **Mix** | 0–100 (wet/dry) | Доля реверба. **0** = сухо; **100** = почти только хвост. Folk/мандо рядом с вокалом — **низкий–умеренный** Mix |
| **Pre Delay** | **ms** | Пауза между **сухой** нотой и началом реверба (ранние отражения / хвост). Больше ms — нота читается отдельно, потом «комната». Мало ms — реверб «липнет» к атаке |
| **Decay** | **0–100** | **Длина** хвоста reverb (не миллисекунды на экране). **Ниже** = короткая комната; **выше** = длиннее «звон» в хвосте. Для folk/chop — низкий–средний Decay |
| **Trail** (Tail) | ON / OFF | При **выключении** Room (байпас / смена патча): **ON** — хвост **доигрывает**; **OFF** — обрыв сразу |

### Decay: что значит «короткий»

На шкале **0–100**:

| Decay | Ощущение |
|-------|----------|
| **10–25** | Маленькая комната, хвост быстро гаснет — folk, чоп, регги |
| **25–40** | Живая комната, ещё не «зал» — баллада, clean |
| **40–60** | Заметный хвост — риск каши с вокалом на плотной сцене |
| **>60** | Ближе к Hall; для Folk Clean обычно много |

---

## Room vs Slapback vs Hall

| | **Slapback** (DLY) | **Room** (RVB) | **Hall** (RVB) |
|--|-------------------|----------------|----------------|
| Суть | одно короткое **эхо** | **размытое** пространство комнаты | большой **зал** |
| Time / Decay | Time в **ms** | Decay **0–100** | Decay длиннее по ощущению |
| Folk Clean | ~100 ms, Mix низкий | Mix умеренный, Decay короткий | на припев вместо Room (CTRL) |

## Примеры

### Folk / вокал / мандокастер

| Параметр | Ориентир |
|----------|----------|
| Mix | **15–22** |
| Pre Delay | **10–25 ms** |
| Decay | **18–30** |
| Trail | **ON** |

См. [Folk Clean](../../../presets/mandocaster/01-folk-clean.md).

### Ещё суше (блюграсс / регги-ритм)

Mix **8–14**, Decay **12–22**, Pre Delay **5–15 ms**.

### Припев чуть «больше воздуха»

Не раздувать Room: переключить на **Hall** с коротким Decay или Room Mix +3…5 — как в Folk Clean CTRL.

---

## Trail ON vs OFF

| | Trail **ON** | Trail **OFF** |
|--|--------------|---------------|
| Байпас mid-фразы | хвост reverb допевает | обрез |
| Смена патча | плавнее | резко сухо |
| Folk Clean | обычно **ON** | если мешает при A/B |

Trail **не** удлиняет reverb во время игры — только при выключении эффекта.

---

## Заметки

- Англ. мануал: *Mix wet/dry; Pre Delay — time before reverb onset; Decay — duration; Trail on bypass.*  
- На сцене с мониторами длинный Decay + высокий Mix → каша и гул; для мандо чаще коротко.  
- DSP: reverb средней нагрузки; при нехватке DSP — упростите CAB/SnapTone.
