# Модули эффектов Valeton GP-180

Цепь по умолчанию:

`NR → PRE → WAH → DST → N→S → AMP → CAB → EQ → MOD → DLY → RVB → VOL`

Модули можно свободно менять местами. Одновременно до 12 слотов.

## Список модулей

- [NR — Noise Gate (шумодав)](./01-nr/) — 3 эффект(ов)
- [PRE — Pre-Effects (пре-эффекты)](./02-pre/) — 23 эффект(ов)
- [WAH — Wah / педаль управления](./03-wah/) — 4 эффект(ов)
- [DST — Distortion / Overdrive](./04-dst/) — 26 эффект(ов)
- [N→S — SnapTone (NAM)](./05-ns/) — 1 эффект(ов)
- [AMP — симуляции усилителей](./06-amp/) — 58 эффект(ов)
- [CAB — кабинеты и IR](./07-cab/) — 36 эффект(ов)
- [EQ — эквалайзеры](./08-eq/) — 4 эффект(ов)
- [MOD — модуляция](./09-mod/) — 17 эффект(ов)
- [DLY — Delay](./10-dly/) — 14 эффект(ов)
- [RVB — Reverb](./11-rvb/) — 11 эффект(ов)
- [VOL — громкость](./12-vol/) — 1 эффект(ов)

**Всего карточек эффектов:** 198
