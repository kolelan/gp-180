# Mess EQ

**Тип:** EQ

## Описание

5-полосный Mesa V-shape: 80 / 240 / 750 / 2.2k / 6.6k.

## Параметры

- Band1–5

## Варианты использования

### Классический V

↑80 и 6.6k, ↓750 — scoopy metal.

### В плотном миксе

Не уходить в крайний scoop — съест гитару под вокалом.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
