# Guitar EQ 1

**Тип:** EQ

## Описание

Гитарный 5-полосный: 125 / 400 / 800 / 1.6k / 4k Гц.

## Параметры

- Band1–5
- Volume

## Варианты использования

### Ритм в миксе

↑800 Гц чуть, ↓125–250 зона баса, ↓4k если скрежет.

### Мандокастер

↓125–400 от гула датчика, ↑1.6k атака, ↓4k резкость.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
