# Empty 1–50 (+user)

**Тип:** SnapTone

## Описание

Слоты SnapTone: заводские 50 + пользовательмые до 100 через Valeton Suite. Параметры как у amp-блока.

## Параметры

- Gain
- VOL
- Bass
- Middle
- Treble
- Presence

## Варианты использования

### Только SnapTone + CAB/IR

AMP OFF, CAB ON или User IR.

### SnapTone без кабинета

Если модель already includes cab — CAB OFF.

### Мандолина

Выбирать open clean/edge-of-breakup, Gain низкий, Presence аккуратно.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
