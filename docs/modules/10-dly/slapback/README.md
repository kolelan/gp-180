# Slapback

**Тип:** Delay *(не reverb — см. [термин](../../../terminy/slapback.md))*

## Описание

Классический слэпбэк: короткий **одиночный** отзвук (эхо), характерный для rockabilly и кантри. Это эффект модуля **DLY**, не зал/комната из RVB.

## Параметры

- Mix
- Feedback
- Time
- Trail

## Варианты использования

### Rockabilly / мандолина

Time ~80–120 ms, Mix низкий.

## Заметки

- Имена производителей оригиналов указаны для описания звукового характера.
- Следите за DSP load цепи: high-precision CAB + SnapTone + тяжёлые FX могут потребовать упрощения.
