# Slapback

**Тип:** Delay *(не reverb — см. [термин](../../../terminy/slapback.md))*  
**Модуль:** [DLY](../)

## Описание

Классический **слэпбэк**: короткий почти **одиночный** отзвук сразу после ноты (rockabilly / кантри / folk). Это модуль **DLY**, не Room/Hall из RVB.

На UI параметр хвоста при байпасе часто подписан **Trail** (в мануале GP-180); иногда воспринимают как «Tail» — это одно и то же.

---

## Параметры

| Параметр | Единицы / тип | На что влияет |
|----------|---------------|----------------|
| **Mix** | 0–100 (wet/dry) | Доля эха в балансе. **0** = только сухой сигнал; **100** = почти только повтор. Для folk/мандо обычно **низкий Mix** — эхо слышно, но не спорит с вокалом |
| **Time** | **ms** (миллисекунды) | Задержка до первого повтора. Классический slapback ~**60–150 ms**. Короче ~80 — «плотный шлёпок»; ~100–120 — чуть просторнее кантри |
| **Feedback** | 0–100 | Сколько сигнала **возвращается** в delay (= сколько раз повторится эхо). **Низкий** Feedback = один (максимум слабый второй) «шлёпок». Высокий = лента повторов / почти зацикливание — это уже не slapback |
| **Trail** (Tail) | ON / OFF | При **выключении** Slapback (байпас / смена патча): **ON** — уже звучащий повтор **доигрывает**; **OFF** — эхо обрывается сразу |

### Что значит «Feedback низкий»

На шкале **0–100**:

| Feedback | Ощущение |
|----------|----------|
| **0–15** | Почти один чёткий повтор — типичный slapback |
| **15–25** | Едва слышен слабый второй «хвостик» — ещё ок для folk |
| **30–50** | Уже заметная серия повторов — ближе к обычному echo |
| **>50** | Длинная лента / риск самовозбуждения — не для Folk Clean |

Для **Folk Clean** ориентир Feedback **~8–18** (старт **12**).

## Примеры

### Folk / вокал / мандокастер

| Параметр | Ориентир |
|----------|----------|
| Mix | **12–18** |
| Time | **90–110 ms** |
| Feedback | **8–18** |
| Trail | **ON** (удобно при смене патча / CTRL) |

### Rockabilly «жирнее»

Mix **20–30**, Time **80–100 ms**, Feedback **10–20**, Trail ON.

### Только воздух, почти не слышно

Mix **8–12**, Time **100 ms**, Feedback **5–12**.

---

## Trail ON vs OFF

| | Trail **ON** | Trail **OFF** |
|--|--------------|---------------|
| Байпас mid-фразы | повтор допевает | эхо рубится |
| Смена патча | хвост может «перетечь» | сухой обрез |
| Folk Clean | обычно **ON** | если мешает при A/B патчей |

Не путать Trail с **Decay реверба**: Trail не удлиняет эхо во время игры — только поведение при выключении эффекта.
